Processo de Solução de Problemas

Etapas do Processo de Solução de Problemas

Após os reparos do computador serem concluídos, finalize o processo de solução de problemas com o cliente. Comunique, verbalmente e por escrito, o problema e a solução para o cliente. A figura apresenta os passos a serem tomados quando você terminar o reparo.

Verifique a solução com o cliente. Se o cliente estiver disponível, demonstre como a solução corrigiu o problema do computador. Peça para o cliente testar a solução e tentar reproduzir problema. Quando o cliente verificar que o problema foi resolvido, você pode completar a documentação para a reparação na ordem de trabalho e em seu diário. Inclua as seguintes informações na documentação:

  • Descrição do problema

  • Passos para resolver o problema

  • Componentes usados ​​no reparo
