Manutenção Preventiva

Visão Geral de Manutenção Preventiva de PC

Hardware

Verifique o estado dos cabos, componentes e periféricos. Limpe os componentes para reduzir a probabilidade de superaquecimento. Repare ou substitua qualquer componente que apresente sinais de danos ou desgaste excessivo.

Use essas tarefas como um guia para a criação de um programa de manutenção de hardware:

  • Remova a poeira das entradas de ventilação.

  • Remova a poeira da fonte de alimentação.

  • Remova a poeira dos componentes dentro do computador e equipamentos periféricos, tais como impressoras.

  • Limpe o mouse, teclado e monitor.

  • Verifique se há cabos soltos.

Software

Verifique se o software instalado é o atual. Siga as políticas da organização ao instalar atualizações de segurança, atualizações do sistema operacional e atualizações de programas. Muitas organizações não permitem atualizações até que os testes sejam concluídos. Este teste é feito para confirmar se a atualização não vai causar problemas com o sistema operacional e o software. A imagem mostra a tela de atualização do Windows 7.

Use essas tarefas como um guia para a criação de um cronograma de manutenção de software que atenda às necessidades do seu computador:

  • Revise e instale as atualizações apropriadas de segurança.

  • Revise e instale as atualizações apropriadas de software.

  • Revise e instale as atualizações apropriadas de drivers.

  • Atualize os arquivos de definição de vírus.

  • Realize uma varredura para procurar vírus e spyware.

  • Remova programas indesejados ou não utilizados.

  • Verifique os discos rígidos para encontrar erros.

  • Desfragmente os discos rígidos não SSD.
