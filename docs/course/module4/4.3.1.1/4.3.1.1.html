Visão Geral de Manutenção Preventiva e Solução de Problemas

Resumo

Este capítulo discutiu sobre os conceitos de manutenção preventiva e os processos de solução de problemas.

  • A manutenção preventiva quando realizada frequentemente reduz os problemas de hardware e software.

  • Antes de iniciar qualquer reparo, faça o backup dos dados em outro computador.

  • O processo de solução de problemas é um guia que o ajudará a resolver os problemas do computador de maneira eficiente.

  • Documente todos os passos realizados, mesmo que não funcionem. A documentação criada será um recurso muito útil para você e outros técnicos.
