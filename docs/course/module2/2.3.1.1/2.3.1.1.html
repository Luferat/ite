Procedimentos de Laboratório e Uso de Ferramentas

Resumo

Este capítulo discutiu procedimentos seguros para laboratórios, o uso da ferramenta correta e o descarte adequado de componentes dos computadores e suprimentos. Você se familiarizou no laboratório com muitas das ferramentas utilizadas para construir, ferramentas de serviço e limpeza de computadores e componentes eletrônicos. Você também aprendeu a importância de ferramentas organizacionais e como estas ferramentas ajudam você a trabalhar com mais eficiência.

Alguns dos conceitos importantes para se lembrar deste capítulo:

  • Trabalhar de uma forma segura para proteger os usuários e equipamentos.

  • Siga todas as orientações de segurança para evitar lesões a si e aos outros.

  • Saiba como proteger o equipamento contra danos ESD.

  • Conhecer e ser capaz de prevenir problemas de energia que podem causar danos aos equipamentos ou perda de dados.

  • Saber quais produtos e suprimentos requerem procedimentos especiais de descarte.

  • Familiarize-se com o MSDS para questões de segurança e restrições de disposição para ajudar a proteger o meio ambiente.

  • Ser capaz de utilizar as ferramentas corretas para cada tarefa.

  • Saber como limpar os componentes de forma segura.

  • Use ferramentas para organização durante os reparos de um computador.
