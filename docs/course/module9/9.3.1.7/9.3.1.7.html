Instalação e Configuração de Impressoras

Instalação e Atualização de Drivers, Firmware e RAM

Depois de instalar uma impressora, imprima uma página de teste para verificar se a impressora está funcionando corretamente. A página de teste confirma que o driver está instalado e funcionando corretamente e que a impressora e o computador estão se comunicando.

Imprimir uma Página de Teste

Para imprimir manualmente uma página de teste no Windows 7, use o seguinte caminho:

Iniciar > Dispositivos e Impressoras (atalho que mostra Dispositivos e Impressoras)

No Windows 7, clique com botão direito na impressora desejada e siga este caminho:

Propriedades da Impressora > Guia Geral > Imprimir Página de Teste

Para imprimir manualmente uma página de teste no Windows Vista, use o seguinte caminho:

Iniciar > Painel de Controle > Impressoras e Aparelhos de Fax

No menu Impressoras e Aparelhos de Fax, clique com o botão direito do mouse na impressora desejada e siga este caminho:

Propriedades > guia Geral > Imprimir Página de Teste

Uma caixa de diálogo pergunta se a página foi impressa corretamente. Se a página não for impressa, os arquivos de ajuda embutidos o ajudará a solucionar o problema.

Para imprimir manualmente uma página de teste no Windows XP, use o seguinte caminho:

Iniciar > Impressoras e Aparelhos de Fax. No menu Impressoras e Aparelhos de Fax, clique com o botão direito do mouse na impressora e selecione Propriedades > Guia Geral > Imprimir Página de Teste

Imprimir a Partir de um Aplicativo

Você pode testar a impressora imprimindo uma página de teste a partir de um aplicativo, como o Bloco de Notas ou o WordPad. Para acessar o Bloco de Notas no Windows 7, Windows Vista e Windows XP, use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Bloco de Notas

No documento em branco, digite algum texto. Para imprimir o documento use o seguinte caminho:

Arquivo > Imprimir

Teste a Impressora

Você pode imprimir a partir da linha de comando para testar a impressora. Impressão a partir da linha de comando é limitada a apenas arquivos ASCII, tais como .txt e .bat.

Para enviar um arquivo para a impressora a partir da linha de comando no Windows 7, clique no botão Iniciar. No campo de pesquisa de programas e arquivos, digite cmd e clique em OK. No prompt de comando, digite o comando Print nome_do_arquivo.txt.

Para enviar um arquivo para a impressora a partir da linha de comando no Windows Vista, use o seguinte caminho:

Iniciar > Iniciar Pesquisa. Na caixa Executar, digite cmd e clique em OK. No prompt de comando, digite o comando Print nome_do_arquivo.txt.

Para enviar um arquivo para a impressora a partir da linha de comando no Windows XP, use o seguinte caminho:

Iniciar > Executar. Na caixa Executar, digite cmd e clique em OK. No prompt de comando, digite o comando Print nome_do_arquivo.txt.

Teste a Impressora no Painel de Impressora

A maioria das impressoras tem um painel frontal com controles para permitir que você gere páginas de teste. Este método de impressão permite que você verifique o funcionamento da impressora separadamente da rede ou do computador. Consulte o site ou a documentação do fabricante para saber como imprimir uma página de teste a partir do painel frontal da impressora.

A instalação de qualquer dispositivo não está completa até que você tenha testado com sucesso todas as suas funções. As funções da impressora podem incluir:

  • Imprimir documentos em frente e verso.

  • Usar diferentes bandejas para diferentes tamanhos de papel.

  • Alterar as configurações de cor de uma impressora para que ela imprima em preto e branco ou escala de cinza.

  • Imprimir no modo rascunho.

  • Usar uma aplicação de reconhecimento óptico de caracteres (OCR).

Para um dispositivo all-in-one, teste todas as funções:

  • Envia um fax para outro aparelho de fax que esteja funcionando.

  • Criar uma cópia de um documento.

  • Digitalizar um documento.

  • Imprimir um documento.

NOTA: Para obter informações sobre como eliminar atolamentos de papel, a instalação de cartuchos de tinta e carregar as bandejas de papel, verifique a documentação ou website do fabricante.
