Instalação e Configuração de Impressoras

Configurações Padrão e Opcionais

Algumas impressoras possuem painéis de controle com botões para selecionar opções. Outras impressoras usam as opções do driver da impressora. Você pode definir as opções globalmente ou por documento.

Método Global

O método global refere-se a opções de impressão que estão definidas para afetar todos os documentos. Cada vez que um documento é impresso, as opções globais são usadas, a menos que sejam substituídas pelas seleções por documento.

Para alterar a configuração global de uma impressora no Windows 7, use o seguinte caminho:

Iniciar> Painel de Controle> Dispositivos e Impressoras> clique com o botão direito do mouse na impressora

Para designar uma impressora padrão, clique com botão direito na impressora e selecione Definir como impressora padrão, como mostrado na imagem.

Para alterar a orientação da página para paisagem no Windows 7, clique com botão direito na impressora e selecione Preferências de impressão. Na guia Layout, selecione paisagem no menu Orientação. Clique em OK.

Para alterar a configuração global de uma impressora no Windows Vista e no Windows XP, use o seguinte caminho:

Iniciar> Painel de Controle> Impressoras e Aparelhos de Fax> clique com o botão direito do mouse na impressora

Para designar uma impressora padrão, clique com o botão direito na impressora e selecione Definir como impressora padrão.

NOTA: Dependendo do driver instalado, a opção Definir impressora padrão pode não estar disponível. Neste caso, dê um duplo clique na impressora para abrir a janela de status do documento e, em seguida, escolha a Impressora > Definir como impressora padrão.

Metodo Por Documento

Cartas, planilhas e imagens digitais são alguns dos tipos de documentos que podem exigir configurações especiais da impressora. Você pode alterar as configurações de um documento individual, alterando as configurações de impressão de documentos.
