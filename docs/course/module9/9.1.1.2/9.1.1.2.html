Recursos Comuns da Impressora

Características e Capacidades

A impressora deve ter uma interface compatível com o computador para que ela possa imprimir. Normalmente, as impressoras se conectam aos computadores domésticos usando uma interface paralela, USB ou sem fio. As impressoras podem se conectar a uma rede usando um cabo ou uma interface wireless.

Serial.

A transferência de dados em série (serial) é o nome dado ao fluxo de cada bit de informação em um único ciclo. Uma conexão serial pode ser utilizada para conectar impressoras matriciais, pois elas não necessitam de altas taxas de transferência de dados.

Paralela

A transferência de dados paralela é mais rápida do que a transferência de dados em série. Uma interface paralela envia múltiplos bits de informação em um único ciclo. O caminho de transferência de dados em uma interface paralela é mais largo do que em uma interface serial, permitindo assim que os dados se movam rapidamente para ou a partir da impressora.

O padrão utilizado pelas portas paralelas de uma impressora é o IEEE 1284. O Enhanced Parallel Port (EPP) e o Enhanced Capabilities Port (ECP) são dois modos de operação do padrão IEEE 1284 que permitem a comunicação bidirecional.

SCSI

O Small Computer System Interface (SCSI) usa a tecnologia de comunicação paralela para atingir altas taxas de transferência de dados.

USB

O USB é uma interface de comunicação de dados comumente utilizada para conectar impressoras e outros dispositivos. Quando um dispositivo USB é adicionado a um computador que suporta plug-and-play, o mesmo é automaticamente detectado e o processo de instalação do driver é iniciado.

FireWire

O FireWire, também conhecido como i.LINK ou IEEE 1394, é um barramento de comunicação de alta velocidade que é independente de plataforma. O FireWire conecta dispositivos digitais, tais como impressoras, scanners, câmeras e discos rígidos.

O FireWire permite que dispositivos periféricos, como uma impressora, se conectem diretamente em um computador. Ele também permite que o dispositivo seja hot-swappable. O FireWire fornece uma única conexão plug-and-socket que pode conectar até 63 dispositivos. Além disso, o FireWire possui uma taxa de transferência de dados de até 400 Mbps.

Ethernet

Conectar uma impressora à rede requer cabeamento compatível tanto com a rede, quanto com a interface instalada na impressora. A maioria das impressoras de rede usam uma interface RJ-45 ou sem fio para se conectar à rede.
