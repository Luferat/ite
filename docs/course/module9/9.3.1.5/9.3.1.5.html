Instalação e Configuração de Impressoras

Instalação e Atualização de Drivers, Firmware e RAM

Os drivers de impressora são programas que tornam possível que computadores e impressoras se comunique uns com os outros. Os softwares de configuração fornecem uma interface que permite aos usuários configurar e alterar as opções da impressora. Cada modelo de impressora possui o seu próprio tipo de driver e software de configuração.

O Page Description Language (PDL) é um código que descreve a aparência de um documento em um idioma para que uma impressora possa entender. O PDL inclui os textos, gráficos e informações de formatação. Um aplicativo de software utiliza um PDL para enviar imagens What You See Is What You Get (WYSIWYG) para a impressora. A impressora traduz o arquivo PDL de modo que tudo o que está na tela do computador será impresso. O PDL acelera o processo de impressão ao enviar grandes quantidades de dados de uma só vez. Eles também gerenciam as fontes do computador.

A Adobe Systems desenvolveu o PostScript para permitir que as fontes e tipos de texto compartilhem as mesmas características da tela, como no papel. A Hewlett-Packard desenvolveu o PCL para comunicação com impressoras jato de tinta antigas. Hoje, o PCL é um padrão da indústria utilizado por quase todos os tipos de impressora.
