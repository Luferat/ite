Sistemas Operacionais Móveis

Características Comuns de Dispositivos Móveis

Orientação da Tela

A maioria das telas dos dispositivos móveis possuem um formato retangular. Este formato permite que o conteúdo a seja visto de duas maneiras diferentes: retrato e paisagem. Alguns conteúdos se encaixam melhor em uma visão específica. Por exemplo, o vídeo preenche toda a tela no modo paisagem, mas ele só preenche metade da tela quando está no modo retrato. Além disso, um livro ou revista visualizado em modo retrato parece muito natural, pois é semelhante a sua forma real. Geralmente, os usuários podem escolher o modo de visualização que é mais confortável para eles em cada tipo de conteúdo.

Muitos dispositivos móveis contêm sensores, como um acelerômetro, que podem determinar como o dispositivo está sendo segurado. Assim, o conteúdo é rodado automaticamente para a posição do aparelho, horizontal ou vertical. Esse recurso é útil ao tirar uma fotografia, por exemplo. Quando o dispositivo está no modo paisagem, o aplicativo da câmera também se transforma em modo paisagem. Além disso, quando um usuário está criando um texto, virar o aparelho para o modo paisagem altera o aplicativo para o modo paisagem, tornando o teclado maior e mais largo, facilitando assim o seu uso.

Quando se utiliza um dispositivo Android, para permitir a rotação automática, conforme mostrado na Figura 1, utilize o seguinte caminho:

Configurações > Tela > selecione Girar tela automaticamente

Quando se utiliza um dispositivo iOS, para permitir a rotação automática, conforme mostrado na Figura 2, utilizar o seguinte caminho:

Pressione duas vezes o botão Home > arraste a barra multitarefa, da esquerda para a direita > pressione o ícone do cadeado, à esquerda

Calibragem da Tela

Ao usar um dispositivo móvel, talvez você possa precisar ajustar o brilho da tela. Aumente o nível de brilho da tela ao usar um dispositivo móvel ao ar livre, pois a luz solar brilhante dificulta a leitura das informações na tela. Por outro lado, um brilho muito baixo é útil ao ler um livro em um dispositivo móvel durante a noite. Alguns dispositivos móveis podem ser configurados para ajustar automaticamente o brilho, dependendo da quantidade de luz ambiente. O dispositivo deve ter um sensor de luz para usar o brilho automático.

A tela de LCD na maioria dos dispositivos móveis utiliza mais energia. Diminuir o brilho ou usar o brilho-automático ajuda a conservar energia. Ajuste o brilho para o nível mais baixo para obter uma maior duração da bateria do dispositivo.

Ao utilizar um dispositivo Android para configurar o brilho da tela, como mostrado na Figura 3, use o seguinte caminho:

Configurações > Tela > Brilho > Ajuste o brilho para o nível desejado ou selecione a opção Brilho Automático > OK

Ao usar um iPad, para configurar o brilho da tela, use o seguinte caminho:

Pressione duas vezes o botão Home > arraste a barra multitarefa, da esquerda para a direita > Ajuste o brilho para o nível desejado > Clique no botão Home

Para configurar o brilho no menu Ajustes, utilize o seguinte caminho:

Ajustes > Imagens de Fundo e Brilho > Ajuste o brilho para o nível desejado ou selecione a opção Brilho Automático > clique no botão Home
