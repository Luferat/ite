Rede de Dados Sem Fio e Celular

Os dispositivos móveis dão às pessoas a liberdade de trabalhar, aprender, jogar e se comunicar a qualquer momento e em qualquer lugar. As pessoas que utilizam dispositivos móveis não precisam estar vinculadas a um local físico para enviar e receber voz, vídeo e dados. Além disso, alguns locais que disponibilizam rede sem fios, tais como cafeterias, estão disponíveis em muitos países. Campus universitários usam redes sem fio para permitir que os alunos se inscrevam para as aulas, assistam a palestras e apresentem trabalhos em áreas onde as conexões físicas para a rede não estão disponíveis. Com os dispositivos móveis cada vez mais poderosos, muitas tarefas que precisam ser realizadas em grandes computadores conectados a redes físicas podem agora ser concluídas através de dispositivos móveis em redes sem fio.

Quase todos os dispositivos móveis são capazes de se conectar à redes Wi-Fi. É aconselhável se conectar nas redes Wi-Fi quando possível, pois os dados usados ​​por Wi-Fi não contam no plano de dados do celular. Além disso, como a interface rádio Wi-Fi utiliza menos energia do que a celular, se conectar em redes Wi-Fi economiza bateria. Como outros dispositivos Wi-Fi, é importante manter a segurança ao se conectar em redes Wi-Fi. Estas precauções devem ser tomadas para proteger as comunicações Wi-Fi em dispositivos móveis:

  • Nunca envie informações de login ou senha usando texto claro não criptografado.

  • Use uma conexão VPN quando possível.

  • Ative a segurança em redes domésticas.

  • Use segurança WPA2.

Para ligar ou desligar o Wi-Fi, utilize o seguinte caminho para o Android, mostrado na Figura 1, e iOS, mostrado na Figura 2:

Configurações > Habilitar ou Desabilitar Wi-Fi

Para conectar um dispositivo Android quando este estiver dentro da faixa de cobertura de uma rede Wi-Fi, ligue o Wi-Fi e, em seguida, procure por todas as redes Wi-Fi disponíveis e exibidas em uma lista. Toque em uma rede Wi-Fi na lista para se conectar. Digite uma senha, se necessário.

Quando um dispositivo móvel está fora do alcance da rede Wi-Fi, ele tenta se conectar a outra rede Wi-Fi no alcance. Se não houver redes Wi-Fi ao alcance, o dispositivo móvel se conecta à rede de dados celular. Quando o Wi-Fi está ativado, ele automaticamente se conecta a qualquer rede Wi-Fi que foi conectada anteriormente. Se a rede for nova, o dispositivo móvel exibirá uma lista de redes disponíveis que podem ser usadas ​​ou perguntará se ele deve se conectar a ela.

Se um dispositivo móvel não solicitar para se conectar a uma rede Wi-Fi, a transmissão do SSID da rede pode estar desligado ou o dispositivo pode não ter sido configurado para se conectar automaticamente. Configure manualmente as definições Wi-Fi no dispositivo móvel.

Para se conectar a uma rede Wi-Fi manualmente em um dispositivo Android, siga estes passos:

Passo 1. Selecione Configurações > Adicionar rede.

Passo 2. Digite o SSID da rede.

Passo 3. Toque em Segurança e selecione o tipo de segurança.

Passo 4. Toque em Senha e digite a senha.

Passo 5. Toque em Salvar.

Para se conectar a uma rede Wi-Fi manualmente em um dispositivo iOS, siga estes passos:

Passo 1. Selecione Ajustes > Wi-Fi > Outras.

Passo 2. Digite o SSID da rede.

Passo 3. Toque em Segurança e selecione o tipo de segurança.

Passo 4. Toque em Outra Rede.

Passo 5. Toque em Senha e digite a senha.

Passo 6. Pressione Conectar.
