Sincronização de Dispositivos Móveis

Para sincronizar os dados de um dispositivo iOS, o software iTunes deve ser instalado, como mostrado na figura. O iTunes é um aplicativo reprodutor de mídia que realiza o download, reproduz e organiza o conteúdo para o uso dos dispositivos iOS e dos computadores. iTunes também gerencia dispositivos iOS, ativando-os com a Apple e restaurando-os quando há um mau funcionamento. O iTunes também é utilizado para atualizar o iOS. Estes são os requisitos de hardware necessários para instalar o iTunes no Windows:

  • PC com processador 1GHz Intel ou AMD

  • 512 MB de RAM

  • Windows XP Service Pack 2 ou edições posteriores, Windows Vista de 32 bits ou Windows 7 de 32 bits

  • As versões 64 bits do Windows Vista ou Windows 7 necessitam do instalador 64 bits do iTunes

Os requisitos mudam de acordo com as versões, além de que o software exige requisitos específicos de áudio e vídeo. Para mais informações, consulte o site da Apple.

Não é necessário nenhum aplicativo para sincronizar o conteúdo dos dispositivos Android. A sincronização automática dos vários tipos de dados e conteúdos do Google é acessada selecionando Configurações> Pessoal> Contas e sincronização.

Utilize o Motocast USB para sincronizar dados e conteúdos ao usar um dispositivo Motorola com o Android OS. Estes são os requisitos de hardware necessários para instalar o iTunes no Windows:

  • PC com processador de 3.0 GHz Intel Pentium IV ou AMD Athlon XP 2600 + processador single-core ou mais rápido ou um PC com um 1.8 GHz Core Duo, processador dual-core ou multi-core Athlon/Phenom ou Pentium mais rápido

  • 1 GB de RAM

  • Windows XP Service Pack 3 ou posterior, Windows Vista ou Windows 7

Especificações adicionais de hardware para áudio ou recursos de vídeo podem ser necessárias. Para mais informações, consulte o site da Motorola.
