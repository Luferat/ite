Métodos de Proteção de Dispositivos Móveis

Serviços de Nuvem em Dispositivos Inteligentes

Se as tentativas de localizar um dispositivo móvel falharem, existem ainda as medidas de segurança que podem ser utilizadas para impedir que os dados do dispositivo sejam comprometidos. Estas medidas podem ser feitas usando o aplicativo Buscar iPhone a partir de um dispositivo iOS ou usando outros aplicativos desenvolvidos para realizar tarefas parecidas. Normalmente, os aplicativos que executam a localização remota possuem medidas de segurança. As duas medidas mais comuns de segurança remota são:

  • Bloqueio Remoto - O recurso de bloqueio remoto, conforme apresentado na Figura 1, permite bloquear um dispositivo utilizando uma senha.

  • Limpeza Remota - O recurso de limpeza remota, conforme apresentado na Figura 2, apaga todos os dados do dispositivo e o retorna ao estado de fábrica. Para restaurar os dados no dispositivo, os usuários do Android devem configurar o dispositivo usando uma conta do Gmail, e os usuários do iOS devem sincronizar seu dispositivo ao iTunes.

NOTA: Para que essas medidas de segurança remotas funcionem, o dispositivo deve estar ligado e conectado a uma rede Wi-Fi ou celular.
