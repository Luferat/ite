Sistemas Operacionais Móveis

Interface de Toque(touch) do Android

Aplicativos

Cada uma das telas iniciais é configurada com uma grade, onde os aplicativos podem ser posicionados. Para mover um aplicativo, siga esses passos:

Passo 1. Toque e segure o aplicativo.

Passo 2. Arraste-o para uma área vazia em qualquer tela inicial.

Passo 3. Solte o aplicativo.

Para remover um aplicativo, a partir de uma tela inicial, siga estes passos:

Passo 1. Toque e segure o aplicativo.

Passo 2. Arraste-o para o ícone da lata de lixo ou para o X localizado na tela.

Passo 3. Solte o aplicativo.

Para executar um aplicativo, toque sobre ele. Durante a execução do aplicativo, existem várias opções que podem ser configuradas ao pressionar o botão menu.

Normalmente, existem três maneiras diferentes para fechar um app:

  • Precione continuamente o botão Voltar até chegar na tela inicial. Geralmente o programa exibe uma imagem confirmando se o usuário pretende sair do aplicativo.

  • Pressione o botão Home.

  • Toque na opção Sair no menu do aplicativo.

Widgets

Widgets são programas (ou partes de programas) que exibem informações. Por exemplo, um widget pode ser colocado na tela inicial para exibir as condições meteorológicas. Muitas vezes, um widget pode ser tocado para executar um aplicativo associado. No caso de um widget de condições climáticas, ao tocá-lo será aberto um aplicativo de tempo em tela cheia, contedo mais detalhes e informações. Widgets são úteis porque eles fornecem acesso rápido a informações ou funções frequentemente usadas. Alguns exemplos de widgets interessantes são os seguintes:

  • Relógio - Exibe uma versão grande de um relógio personalizável.

  • Tempo - Mostra as condições atuais para um ou mais locais.

  • Wi-Fi On/Off - Permite que o usuário ligue ou desligue o Wi-Fi rapidamente, sem precisar navegar até o menu de configurações.

  • Controle Energia - Apresenta vários widgets, como Wi-Fi On/Off, Bluetooth e o modo vibrar em um grupo de Widgets que está sempre visível.

  • Notas - Permite ao usuário criar notas que são exibidas na tela inicial.

  • Facebook - Fornece acesso ao Facebook e permite ao usuário postar no mesmo rapidamente.

  • Ligações - Exibe a última chamada recebida ou realizada.

Existem muitos outros tipos de elementos que podem ser utilizados para personalizar as telas Android. Consulte a documentação de programas ou do Google Play para determinar quais aplicativos possuem widgets.

Pastas

Em alguns dispositivos móveis, vários aplicativos podem ser agrupados em pastas para melhor organizá-los. Se opções de pastas não estiverem disponíveis, um aplicativo pode ser instalado para fornecer esse recurso. Os aplicativos podem ser agrupados de qualquer forma desejada.

Para criar uma pasta em um smartphone Android, siga esses passos:

Passo 1. Toque e segure um aplicativo na tela inicial.

Passo 2. Arraste o aplicativo para a pasta.

Passo 3. Solte o aplicativo.

Para criar uma pasta em um tablet Android, siga esses passos:

Passo 1. Toque e segure um aplicativo na tela inicial.

Passo 2. Arraste o ícone do aplicativo para a pasta, localizada na parte superior da tela.

Passo 3. Solte o aplicativo.

Toque em qualquer pasta para abri-la. Toque em qualquer aplicativo dentro da pasta para abri-lo. Para renomear uma pasta, toque na pasta, toque no nome da pasta e digite um novo nome. Para sair de uma pasta, toque fora da pasta ou pressione o botão Home ou Voltar. Você pode mover a pasta de uma tela para outra da mesma forma que você move um aplicativo.

Para remover um aplicativo de uma pasta, siga esses passos:

Passo 1. Abra a pasta.

Passo 2. Toque e segure o aplicativo a ser removido.

Passo 3. Arraste o aplicativo para uma área vazia da tela inicial.

Passo 4. Solte o aplicativo.

Em smartphones Android, se a pasta possuir dois aplicativos e um deles é removido, a pasta é excluída, porém o aplicativo restante substitui a pasta na tela inicial. Em tablets Android, se todos os aplicativos são removidos de uma pasta, a pasta não é excluída. Você pode remover a pasta de uma tela da mesma forma que você remove um aplicativo.

Ícone de Aplicativos

O ícone Aplicativos exibe todos os aplicativos que estão instalados no dispositivo. Estas são algumas das tarefas comuns que podem ser executadas nessa tela:

  • Executar um aplicativo - Toque em qualquer aplicativo para iniciá-lo.

  • Mover aplicativo para a tela inicial - Toque e segure um aplicativo. A tela inicial será exibida. Solte o aplicativo em qualquer área livre de qualquer tela inicial.

  • Desinstalar aplicativos - Toque e segure um aplicativo. Arraste o aplicativo para o ícone da lixeira ou para o ícone X.

  • Acesse o Play Store - Toque no ícone do menu e depois toque no ícone Google Play.

  • Widgets - Toque na aba Widgets para visualizar todos os widgets instalados. Adicione os widgets na tela inicial usando o mesmo método necessário para adicionar aplicativos na tela inicial.
