Métodos de Proteção de Dispositivos Móveis

Senhas de Bloqueio

Smartphones, tablets e outros dispositivos móveis contêm dados sensíveis. Se um dispositivo móvel for perdido, quem encontrar o dispositivo terá acesso aos contatos, mensagens de texto e as contas da web. Um método para evitar o roubo de informações privadas a partir de dispositivos móveis é usar um código de bloqueio. O código de bloqueio bloqueia um dispositivo e coloca o mesmo em um estado de economia de energia. O bloqueio também pode ser adiado para ocorrer após um determinado período de tempo depois que o dispositivo entrou no estado de economia de energia. Um método comum para colocar um dispositivo móvel em um estado de economia de energia é pressionar rapidamente o botão de alimentação principal. O dispositivo também pode ser configurado para entrar no estado de sleep depois de uma certa quantidade de tempo.

Muitos tipos diferentes de códigos de bloqueio estão disponíveis, conforme apresentado na Figura 1. Alguns tipos de códigos são mais difíceis de adivinhar do que outros. O código deve ser digitado toda vez que o dispositivo é ligado ou sai de um estado de economia de energia. Estes são alguns dos tipos mais comuns de códigos de bloqueio:

  • Nenhum - Remove qualquer outro tipo de código de bloqueio configurado.

  • Deslizar - O utilizador deslizar um ícone, tal como um cadeado ou uma seta, para desbloquear o dispositivo. Esta opção é o menos segura.

  • Desbloqueio facial - Usa a câmera para reconhecer o rosto. Depois que o rosto armazenado é reconhecido, o dispositivo é desbloqueado.

  • Padrão - Bloqueia o dispositivo quando o usuário desliza o dedo sobre a tela, desenhando um determinado padrão. Para desbloquear o dispositivo, o padrão exato deve ser repetido na tela.

  • PIN - Usa um PIN privado para proteger o dispositivo. Quando o pin é digitado corretamente o é aparelho desbloqueado.

  • Senha - Usa uma senha para proteger o dispositivo. Esta opção é menos conveniente, especialmente se a palavra for complicada ou longa, mas pode ser o método mais seguro.

  • Código Simples - Apenas para dispositivos iOS. Quando esta opção é configurada, o código deve ser um número de quatro dígitos. Quando esta opção está desativada, as senhas mais complexas que usam caracteres, números e símbolos podem ser usadas.

Depois que o código de bloqueio é definido, ele deve ser digitado sempre que o dispositivo for ligado ou sair de um estado de economia de energia.

Para definir um código em um dispositivo Android, use o seguinte caminho:

Configurações > Local e Segurança > Bloqueio de Tela Escolha o tipo de senha para usar e defina as configurações de segurança restantes.

Para configurar um código de acesso em um dispositivo iOS, use o seguinte caminho:

Ajustes > Geral > Bloqueio por Código > Ativar Código. Digite um código de quatro dígitos, conforme apresentado na Figura 2. Introduza o mesmo número na segunda vez para verificação.

Se o código de acesso para um dispositivo iOS for esquecido, você deve conectá-lo ao computador para o qual foi sincronizado pela última vez e realizar uma restauração via iTunes.

Você também pode executar uma restauração de um dispositivo Android. Isso é feito pressionando os botões de volume ao ligar o aparelho, depois que uma opção de restauração é exibida. Verifique com o fabricante do seu dispositivo Android para obter instruções específicas.
