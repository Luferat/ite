Interface de Toque do iOS

A interface iOS funciona de uma forma muito similar a interface Android. As telas iniciais são usadas ​​para organizar os aplicativos, que são executados com o toque. Existem algumas diferenças muito importantes:

  • Não há ícones de navegação - Para acessar a tela inicial, um botão físico deve ser pressionado, ao invés de tocar em ícones de navegação.

  • Sem Widgets - Apenas aplicativos e outros conteúdos podem ser instalados em dispositivos iOS.

  • Não há atalhos de aplicativos - Os ícones exibidos na tela inicial representem o aplicativo real e não um atalho.

Botão Home

Diferente do Android, dispositivos iOS não usam ícones de navegação para executar funções. Um único botão chamado o botão Home executa muitas das mesmas funções que os botões de navegação para Android realizam. O botão home está localizado na parte inferior do dispositivo, conforme apresentado na Figura 1. O botão Home executa as seguintes funções:

  • Despertar o dispositivo - Quando a tela estiver desligada, pressione o botão Home para ativá-la.

  • Retornar à tela inicial - Pressione o botão Home, enquanto estiver usando um app, para acessar a última tela inicial que foi usada.

  • Retornar à tela inicial principal - Pressione o botão Home, enquanto estiver em qualquer outra tela inicial ou na tela de busca para ir para a tela inicial principal.

  • Abrir a barra de multitarefa - Pressione duas vezes o botão Home para abrir a barra de multitarefa. A barra de multitarefa mostra os aplicativos mais recentes que têm sido utilizados.

  • Iniciar o controle por voz Siri - Segure o botão home para iniciar o controle de voz Siri. O Siri é um software especial que entende comandos de voz avançados. Em dispositivos que não possuem Siri, os controles básicos de voz podem ser utilizados.

  • Controles de áudio - Pressione duas vezes o botão Home quando a tela estiver bloqueada para abrir os controles de áudio.

  • Abrir a tela de pesquisa - Pressione o botão Home, enquanto estiver na tela inicial principal para ir para a tela de pesquisa.

Centro de Notificações

Dispositivos iOS possuem um centro de notificação que exibe todos os alertas de aplicativos em um único local, conforme apresentado na Figura 2. Para abrir a área de notificação em dispositivos iOS, toque e segure a parte superior da tela e arraste o dedo para baixo. As seguintes ações podem ser tomadas quando as notificações estiverem abertas:

  • Responder a um alerta tocando-o.

  • Remover um alerta ao tocar no ícone X.

Para alterar as opções de notificação, use o seguinte caminho:

Ajustes > Central de Notificações

Pesquisa

Na tela inicial principal, arraste o dedo da esquerda para a direita ou pressione duas vezes o botão Home para abrir o recurso de pesquisa, chamado Spotlight, conforme apresentado na Figura 3. Digite o item de pesquisa. Todos os resultados são listados de acordo com a aplicação que contém os resultados. Para realizar uma pesquisa na Internet, toque em pesquisa na web, localizado na parte inferior da lista, depois que os resultados da pesquisa forem exibidos.
