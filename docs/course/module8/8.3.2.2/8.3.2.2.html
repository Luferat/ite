Bluetooth

Emparelhamento Bluetooth acontece quando dois dispositivos Bluetooth estabelecem uma conexão para compartilhar recursos. Para que os dispositivos possam emparelhar-se, o Bluetooth deve estar ativo e um dispositivo deve começar a procurar outros dispositivos. Os outros dispositivos devem estar configurados para o modo detectável, também chamado visível, de modo que possam ser detectados. Quando um dispositivo Bluetooth está no modo visível, ele transmite as seguintes informações quando o outro dispositivo Bluetooth as solicita:

  • Nome

  • Classe Bluetooth

  • Serviços que podem usar o dispositivo

  • Informações técnicas, como as características ou especificações que são suportadas pelo Bluetooth

Durante o processo de emparelhamento, pode ser requisitado um PIN para autenticar o processo de emparelhamento, como mostrado na Imagem 1. O PIN muitas vezes é um número, mas também pode ser um código numérico ou senha. O PIN é armazenado usando os serviços de emparelhamento, desta forma ele não tem que ser introduzido na próxima vez que o dispositivo tentar se conectar. Isto é conveniente quando se usa um fone de ouvido com um smartphone, pois eles são emparelhados automaticamente quando o fone de ouvido está ligado e dentro do alcance.

Para emparelhar um dispositivo Bluetooth com um dispositivo Android, siga estes passos:

Passo 1. Siga as instruções do seu dispositivo para colocá-lo em modo de descoberta.

Passo 2. Verifique as instruções do seu dispositivo para encontrar o PIN da conexão.

Passo 3. Selecione Configurações> Redes sem fio e outras.

Passo 4. Toque em Bluetooth para ativá-lo.

Passo 5. Toque na guia Bluetooth.

Passo 6. Toque em Procurar dispositivos.

Passo 7. Toque no dispositivo descoberto para selecioná-lo.

Passo 8. Digite o PIN.

Passo 9. Toque no nome do dispositivo para se conectar a ele.

Para emparelhar um dispositivo Bluetooth com um dispositivo iOS, siga estes passos:

Passo 1. Siga as instruções do seu dispositivo para colocá-lo em modo de descoberta.

Passo 2. Verifique as instruções do seu dispositivo para encontrar o PIN da conexão.

Passo 3. Selecione Configurações> Geral> Bluetooth.

Passo 4. Toque em Bluetooth para ativá-lo.

Passo 5. Toque no dispositivo descoberto para selecioná-lo.

Passo 6. Digite o PIN.
