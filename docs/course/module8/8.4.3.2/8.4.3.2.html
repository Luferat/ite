Métodos de Proteção de Dispositivos Móveis

Software de Segurança

Como o sistema operacional de um desktop ou laptop, você pode atualizar ou corrigir o sistema operacional de dispositivos móveis. Atualizações adicionam funcionalidades ou melhoram o desempenho. Os patches podem corrigir problemas de segurança ou de algum hardware ou software que não esteja funcionando corretamente.

Pelo fato de existirem diferentes dispositivos móveis Android as atualizações e patches não são lançadas como um pacote para todos os dispositivos. Às vezes, uma nova versão do Android não pode ser instalada em dispositivos mais antigos, onde o hardware não atende às especificações mínimas. Estes dispositivos podem receber patches para corrigir problemas conhecidos, mas não receberão atualizações do sistema operacional.

Atualizações e patches Android usam um processo automatizado para a entrega. Quando uma operadora ou fabricante possui uma atualização para um dispositivo, ela é apresentada como uma notificação no dispositivo informando que há uma atualização. Toque na atualização para iniciar o processo de download e instalação.

As atualizações do iOS utilizam um processo automatizado para entrega e os dispositivos que não atendem aos requisitos de hardware são excluídos. Para verificar se há atualizações para iOS, conecte o dispositivo ao iTunes. Um aviso de download é indicado se há atualizações disponíveis. Para verificar manualmente por atualizações, clique no botão Check for Update no painel de Resumo do iTunes.
