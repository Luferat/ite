Dispositivos Móveis

Resumo

Este capítulo introduziu-lhe aos dispositivos móveis, aos sistemas operacionais usados ​​por dispositivos móveis, como proteger os dispositivos móveis, o uso de serviços cloud-enabled e a forma com que os dispositivos móveis se conectam à rede, dispositivos e periféricos. Conceitos básicos de solução de problemas relacionados à dispositivos móveis foram discutidos, apresentando exemplos de soluções simples para problemas comuns. Os seguintes conceitos deste capítulo são importantes para serem lembrados:

  • O hardware dos dispositivos móveis possui poucas unidades reparáveis em campo.

  • Os dispositivos móveis frequentemente são substituídos ao invés de reparados, devido ao alto custo dos consertos.

  • Os dispositivos móveis muitas vezes contêm peças exclusivas que não podem ser trocadas.

  • Os touchscreens são usados ​​no lugar de outros dispositivos de entrada, tais como mouse e teclado.

  • Discos SSDs são usados ​​em dispositivos móveis devido ao seu tamanho, eficiência energética e funcionamento silencioso.

  • Os softwares de código aberto podem ser modificados por qualquer pessoa com pouco ou nenhum custo.

  • Utilize apenas ​​fontes de conteúdo confiáveis para evitar malwares e conteúdos inseguros.

  • Tanto Android, quanto o iOS possuem GUIs semelhante para usar aplicativos e acessar outros conteúdos.

  • Os dispositivos móveis utilizam sensores, como GPS e acelerômetros, para melhorar a sua funcionalidade.

  • As conexões de rede para dispositivos móveis são feitas através das redes celular, Wi-Fi e Bluetooth.

  • As contas de e-mail ficam atreladas aos dispositivos móveis, fornecendo diversos serviços de sincronização de dados.

  • Dispositivos Android usam aplicativos para sincronizar os dados que não são sincronizados automaticamente pelo Google.

  • Dispositivos iOS usam o iTunes para sincronizar dados e outros conteúdos.

  • Padrões de segurança e senhas são usados para protegem os dispositivos móveis.

  • Backups remotos podem ser realizados para armazenar dados dos dispositivos móveis na nuvem.

  • Bloqueio ou o apagamento remoto são recursos usados para proteger um dispositivo móvel que foi perdido ou roubado.

  • Softwares de antivírus são frequentemente usados em dispositivos móveis para impedir a transferência de programas maliciosos para outros dispositivos ou computadores.
