Conexão de Rede de Computadores

Configuração dos roteadores com e sem fio

Assim que o roteador tiver estabelecido a comunicação com o modem, configure o mesmo para se comunicar com os dispositivos na rede. Abra navegador web. No campo de endereço, digite 192.168.1.1. Este é o endereço IP privado padrão utilizado para configurar o roteador Linksys E2500, através da interface de gerenciamento.

Na primeira vez que o usuário se conectar no roteador Linksys E2500, será solicitado a instalação do software Cisco Connect. Caso o usuário não queira instalar esta ferramenta, ele poderá acessar o dispositivo manualmente através do navegador. Ao conectar-se manualmente em um roteador E2500, uma janela de de segurança, conforme exibida na figura, solicitará que o usuário se autentique para que ele possa acessar as telas de configuração. O campo username deve ser deixado em branco. Digite admin (senha padrão) no campo de senha.
