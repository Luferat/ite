Energia do Laptop

Configurações de Energia

Frequentemente é solicitado aos técnicos que configurem as definições de energia, alterando as configurações no setup da BIOS. A imagem apresenta um exemplo de configuração de energia no setup da BIOS. Definir as configurações de energia no setup da BIOS afeta os seguintes aspectos:

  • Estado do sistema

  • Bateria e modo de AC (corrente alternada)

  • Gerenciamento térmico

  • Gerenciamento de energia do barramento PCI CPU

  • Wake on LAN (WOL)

NOTA: WOL pode exigir uma conexão a cabo de dentro do computador, do adaptador de rede para a placa-mãe.

No Windows, o modo de gerenciamento de energia ACPI deve ser ativado nas configuração da BIOS para permitir que o sistema operacional configure os estados de gerenciamento de energia.

Para ativar o modo ACPI no setup da BIOS, siga estes passos:

Passo 1. Entre no setup da BIOS.

Passo 2. Localize e insira o item Gerenciamento de Energia no menu de configurações.

Passo 3. Use as teclas apropriadas para ativar o modo ACPI.

Passo 4. Salve e saia do setup da BIOS.

NOTA: Estes passos são comuns à maioria dos laptops, mas não se esqueça de verificar a documentação do laptop para configurações específicas. Não há nenhum nome padrão para cada estado de gerenciamento de energia. Os fabricantes podem usar nomes diferentes para o mesmo estado.
