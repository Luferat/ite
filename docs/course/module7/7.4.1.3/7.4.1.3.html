Tecnologias de Comunicação Sem Fio de Laptops

Recursos e Configurações do Sistema Operacional

Para conectar um laptop a uma WAN de celular, é necessário instalar um adaptador projetado para enviar dados neste tipo de rede. Adaptadores WAN para redes celular, conforme apresentado na Figura 1, são dispositivos plug-and-play (PnP). Quando este adaptador não é embutido no hardware do laptop, é necessário conectar as placas adaptadoras no slot PC Card. A WAN celular também pode ser acessada através do uso de um adaptador USB ou de um hotspot móvel, conforme apresentado na Figura 2. O hotspot móvel pode ser conectado ao laptop usando a tecnologia Wi-Fi. Alguns telefones celulares também oferecem funcionalidade hotspot móvel.

Para os laptops que possuem recursos de WAN celular integrado, não é necessário instalar nenhum software, antena ou acessório adicional para fazer uso deste recurso. Ao ligar o laptop, os recursos de WAN integrado já estão prontos para serem usados. Se a conexão WAN celular não estiver ativa, procure por um botão na parte frontal ou lateral do laptop para ativá-la, seguindo os seguintes passos:

Passo 1. Antes de instalar e configurar o dispositivo, certifique-se de que a WAN celular está habilitada na BIOS.

Passo 2. Instale o software utilitário da placa WAN celular do fabricante.

Passo 3. Utilize o utilitário do software para gerenciar a conexão de rede.

O software utilitário da WAN celular geralmente está localizado na barra de tarefas ou em Iniciar > Programas.
