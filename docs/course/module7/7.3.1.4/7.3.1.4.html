Energia do Laptop

Configurações de Energia

O utilitário de Opções de Energia do Windows permite que você reduza o consumo de energia de dispositivos específicos ou de todo o sistema. Você pode gerenciar o uso de energia para o seguinte:

  • Laptop

  • Disco rígido.

  • Tela

  • Suspensão

  • Avisos de bateria fraca

Para configurar as definições de energia no Windows 7 e Windows Vista, use o seguinte caminho:

Iniciar > Painel de Controle > Opções de Energia

Para definir as configurações de energia do Windows XP, use o seguinte caminho:

Iniciar > Painel de Controle > Opções de Energia

Opções de Energia do Laptop

Se você não quiser desligar completamente o laptop quando você pressionar o botão de energia, você pode ajustar as configurações para reduzir o consumo de energia.

Para acessar a definição dos botões de energia e ativar o menu de proteção por senha no Windows 7 e Windows Vista, clique no link "Escolher a função dos botões de energia", localizado a esquerda do utilitário Opções de Energia. No Windows XP, as configurações de botões de energia podem ser acessadas selecionando a guia Avançado no utilitário Opções de Energia.

No Windows 7 e Windows Vista, as opções são:

  • Não fazer nada - O computador continua a funcionar a plena potência.

  • Suspender - Documentos e aplicativos são salvos na memória RAM, permitindo que o computador ligue rapidamente.

  • Hibernar - Documentos e aplicativos são salvos em um arquivo temporário no disco rígido. O laptop demora um pouco mais que a Suspensão para ligar.

No Windows XP, as opções são:

  • Espera - Documentos e aplicativos são salvos na memória RAM, permitindo que o computador ligue rapidamente.

  • Hibernar - Documentos e aplicativos são salvos em um arquivo temporário no disco rígido. O laptop demora um pouco mais que o modo de Espera para ligar.

Gerenciamento de Energia do Disco Rígido e da Tela

Dois dos maiores consumidores de energia em um laptop são o disco rígido e a tela. Conforme mostrado na Figura 2, você pode selecionar quando desligar o disco rígido ou a tela se o laptop estiver funcionando com bateria ou no adaptador de energia.

Para ajustar as configurações de energia para a unidade de disco rígido, tela ou outro componente do computador no Windows 7 e Windows Vista, siga estes passos:

Passo 1. Clique em Iniciar > Painel de Controle > Opções de Energia.

Passo 2. Localize o plano de energia.

Passo 3. Clique em Alterar configurações do plano.

Passo 4. Clique em Alterar configurações de energia avançadas.

Para acessar as configurações de energia avançadas no Windows XP, use o seguinte caminho:

Iniciar > Painel de Controle > Opções de Energia > guia Avançado

Para configurar os temporizadores de Suspensão no Windows 7 e Windows Vista, siga estes passos:

Passo 1. Clique em Iniciar > Painel de Controle > Opções de Energia.

Passo 2. Clique em Alterar quando o computador é suspenso e selecione o tempo desejado.

Para configurar os temporizadores de suspensão no Windows XP, use o seguinte caminho:

Iniciar > Painel de Controle > Opções de Energia e selecione o tempo desejado

Avisos da Bateria

No Windows, você pode definir os níveis de avisos da bateria. O padrão para um aviso de bateria fraca é de 10 porcento da capacidade restante. O padrão para o nível crítico da bateria é de 5 porcento. Você também pode definir o tipo de notificação e a ação a ser tomada, seja para suspender, hibernar ou desligar o laptop quando a capacidade da bateria atingir o nível especificado.
