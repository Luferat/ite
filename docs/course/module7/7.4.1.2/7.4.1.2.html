Tecnologias de Comunicação sem Fio de Laptops

Recursos e Configurações do Sistema Operacional

O infravermelho (IR) é uma tecnologia sem fio de curto alcance e baixa potência. Os infravermelhos transmitem dados utilizando LEDs e recebem através de fotodiodos. Redes sem fio infravermelho são globalmente regulamentadas. No entanto, a Infrared Data Association (IrDA) define as especificações para a comunicação IR sem fio. A Figura 1 apresenta características comuns do Infravermelho.

Existem três tipos comuns de redes Infravermelho:

  • Linha de visão - O sinal só é transmitido se houver uma visão clara e desobstruída entre os dispositivos.

  • Dispersão – O sinal é refletido no teto e na parede.

  • Reflexiva - O sinal é enviado para um transceptor óptico e é reencaminhado para o dispositivo receptor.

Instalar e configurar dispositivos Infravermelho é simples. Muitos dispositivos infravermelho conectam-se à porta USB de um laptop ou desktop. Quando o computador detecta o novo dispositivo, o Windows 7 instala os drivers apropriados. A instalação é semelhante a configurar uma conexão LAN.

O infravermelho é uma solução de conexão prática, de curto alcance, mas que tem algumas limitações:

  • A luz do infravermelho não pode penetrar tetos ou paredes.

  • Os sinais infravermelho são susceptíveis a interferência e diluição por fortes fontes de luz, tais como luzes fluorescentes.

  • Os dispositivos infravermelhos de dispersão conseguem estabelecer uma conexão sem uma linha de visão, mas a velocidade de transferência de dados é inferior e as distâncias são mais curtas.

  • As distâncias dos dispositivos infravermelho devem ser de 3 pés (1 m) ou inferiores quando utilizadas para comunicações de computador.

Antes de instalar e configurar um dispositivo, certifique-se que o infravermelho está habilitado na BIOS, seguindo estes passos:

Passo 1. Ligue o dispositivo, tornando-o visível para Windows.

Passo 2. Alinhe os dispositivos.

Passo 3. Quando os dispositivos estiverem corretamente alinhados, um ícone aparecerá na barra de tarefas com uma mensagem pop-up.

Passo 4. Clique na mensagem para exibir a caixa de diálogo do infravermelho.

Laptops que não possuem um dispositivo infravermelho interno podem se conectar em um transceptor infravermelho através de uma porta serial ou USB. A Figura 2 exibe um transceptor infravermelho utilizando uma porta interna.

Você também pode acessar a caixa de diálogo do infravermelho no Painel de Controle. Você pode configurar as seguintes definições:

  • Infravermelho - Controla a forma como você é notificado sobre uma conexão infravermelho e controla como os arquivos são transferidos.

  • Transferência de Imagem - Controla como as imagens são transferidas de uma câmara digital.

  • Hardware - Lista os dispositivos de infravermelho que são instalados no computador.
