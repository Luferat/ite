Aplicar o Processo de Solução de Problemas em Laptops

Problemas de laptops podem ser resultado de uma combinação de problemas de hardware, software e rede. Os técnicos devem ser capazes de analisar o problema e determinar a causa do erro, para então reparar o laptop. Este processo é chamado de troubleshooting (solução de problemas).

O primeiro passo no processo de solução de problemas é identificar o problema. A figura apresenta uma lista de questões que devem ser feitas aos clientes.
