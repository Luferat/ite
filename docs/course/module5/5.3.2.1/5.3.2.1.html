Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

O Windows centraliza as configurações de muitos recursos que controlam o comportamento e a aparência do computador. Essas configurações são categorizadas em utilitários, ou pequenos programas, encontrado no Painel de Controle, como mostrado na Imagem 1. Adicionar ou remover programas, alterar configurações de rede e alterar as configurações de segurança são algumas das opções de configuração disponíveis no Painel de Controle.

Os nomes das diferentes categorias no painel de controle variam um pouco, dependendo da versão do Windows instalada. Por padrão, os ícones são agrupados nas seguintes categorias no Windows 7:

  • Sistema e Segurança - Utilizado para configurar as definições de segurança e do sistema

  • Rede e Internet - Utilizado para configurar os tipos de conexão de rede

  • Hardware e Sons - Utilizado para configurar os dispositivos conectados ao computador e as configurações de som

  • Programas - Utilizado para remover, alterar e reparar aplicativos

  • Contas de Usuário e Segurança Familiar - Utilizado para criar e remover contas de usuário e configurar os controles dos pais

  • Aparência e Personalização - Utilizado para controlar a aparência da interface de usuário do Windows

  • Relógio, Idioma e Região - Utilizado para especificar a localização e o idioma

  • Facilidade de Acesso - Utilizado para configurar o Windows conforme as necessidades visuais, auditivas e motoras

No Windows, você pode alterar a forma em que o Painel de Controle é exibido. O tipo de visualização que você selecionar determina quais utilitários podem ser acessados imediatamente no Painel de Controle. No Windows 7, as opções de exibição incluem:

  • Categoria - Conjuntos de utilitários do Painel de Controle em grupos de fácil navegação.

  • Ícones Grandes - Exibe os utilitários em ordem alfabética, usando ícones grandes.

  • Ícones Pequenos - Exibe os utilitários em ordem alfabética, usando ícones pequenos.

NOTA: Este curso utiliza a opção "Ícones Grandes", como mostrado na Imagem 2.

No Windows Vista, há duas opções de visualização:

  • Página Inicial do Painel de Controle - Conjunto de utilitários do Painel de Controle em grupos de fácil navegação.

  • Modo de Exibição Clássico - Exibe todos os utilitários do Painel de Controle individualmente.

No Windows XP, existem duas opções de visualização:

  • Categoria - Conjuntos de utilitários do Painel de Controle em grupos de fácil navegação.

  • Modo de Exibição Clássico - Exibe todos os utilitários do Painel de Controle individualmente.
