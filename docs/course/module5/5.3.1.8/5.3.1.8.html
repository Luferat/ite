Interface Gráfica do Windows e Painel de Controle

Desktop, Ferramentas e Aplicações

Computador

O recurso Computador permite que você acesse várias unidades instaladas no PC. No o Windows 7 ou Windows Vista, clique em Iniciar e selecione Computador. No Windows XP, esse recurso é chamado de Meu computador, e você pode acessá-lo clicando duas vezes no ícone Meu computador na área de trabalho.

Windows Explorer

O Windows Explorer permite que você navegue no sistema de arquivos. O painel de navegação do Windows 7, conforme apresentado na figura, inclui o caminho da pasta ou arquivo acessado atualmente no topo, uma barra de busca no canto superior direito, uma barra de ferramentas embaixo da listagem do caminho e da barra de pesquisa, um painel de navegação à esquerda e um painel de visualização à direita do painel de navegação. Você pode usar a barra de busca para encontrar um aplicativo específico, arquivo ou pasta. A barra de ferramentas permite que você organize os arquivos, adicione novas pastas, altere o layout de arquivos no Windows Explorer, exiba um painel de pré-visualização de arquivos e pastas, além de abrir o recurso de ajuda do Windows. O painel de navegação inclui uma lista padrão de pastas, incluindo os Favoritos, Bibliotecas, Computador e Rede. Você pode usar o painel de visualização para acessar ou manipular aplicativos, arquivos e pastas.

O windows 7 adiciona as seguintes funcionalidades para o Windows Explorer:

  • Caixa de Pesquisa - Acessa pesquisas anteriores. Você também pode filtrar os resultados das buscas.

  • Nova pasta - Cria novas pastas com apenas um clique.

  • Organizar por: - Organiza os itens facilmente com diferentes critérios.

  • Botão do painel de visualização - Habilita ou desabilitada o painel de visualização.

Iniciando Aplicativos e Abrindo Arquivos

Você pode iniciar um aplicativo de várias maneiras:

  • Clique no ícone de um aplicativo no Menu Iniciar ou na barra de tarefas.

  • Dê um duplo clique no executável do aplicativo ou no ícone de atalho localizado no desktop ou no Windows Explorer.

  • Inicie o aplicativo a partir da janela Executar ou da console de linha de comando.

Você pode abrir os arquivos da mesma forma que as aplicações. Quando você abre um arquivo, o Windows determina qual aplicativo está associado com o arquivo. O Windows compara a extensão do arquivo com as aplicações instaladas capazes de abrir o arquivo. Por exemplo, se você abrir um arquivo docx., O Windows iniciará o Microsoft Word e abrirá o arquivo.
