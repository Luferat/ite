Interface Gráfica do Windows e Painel de Controle

Desktop, Ferramentas e Aplicações

O Menu Iniciar e a Barra de Tarefas permitem aos usuários gerenciar os programas, procurar no computador e manipular aplicativos em execução. Para personalizar o Menu Iniciar ou a Barra de Tarefas, clique com o botão direito do mouse e escolha Propriedades.

Menu Iniciar

No desktop, o Menu Iniciar é acessado clicando no ícone do Windows, no canto inferior esquerdo da área de trabalho. O Menu Iniciar, mostrado na Imagem 1, exibe todas as aplicações instaladas no computador, uma lista de documentos abertos recentemente e uma lista de outros elementos, como o recurso de pesquisa, central de ajuda e configurações do sistema. Você pode personalizar diferentes aspectos do Menu Iniciar, como mostrado na Imagem 2. Para personalizar as configurações do menu Iniciar, use o seguinte caminho:

Clique com o botão direito do mouse em uma parte vazia da barra de tarefas e escolha Propriedades> Menu Iniciar> Personalizar

No Windows XP, você pode mudar o estilo do Menu Iniciar: XP ou Classic. No Windows 7, a opção de estilos do Menu Iniciar foi removida.

Barra de Tarefas

No Windows 7, os seguintes recursos foram adicionados à barra de tarefas para tornar a navegação mais organização e o acesso a janelas e notificações mais fácil:

  • Lista de Atalhos - Para exibir uma lista de tarefas que são exclusivas para o aplicativo, clique com o botão direito do mouse no ícone do aplicativo na barra de tarefas.

  • Aplicativos Fixados - Para adicionar um aplicativo na barra de tarefas e facilitar o seu acesso, clique com o botão direito do mouse no ícone de um aplicativo e selecione a opção Fixar na Barra de Tarefas.

  • Visualizações em Miniatura - Para ver uma imagem em miniatura de um programa em execução, passe o mouse sobre o ícone do programa na barra de tarefas.
