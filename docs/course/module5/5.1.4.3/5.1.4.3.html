Sistemas Operacionais Modernos

Atualizações do Sistema Operacional

Quando uma nova instalação for necessária, os dados do usuário devem ser migrados do sistema operacional antigo para o novo. Existem três ferramentas disponíveis para realizar a transferência dos dados e das configurações. A ferramenta que você escolherá depende do seu nível de experiência e de suas necessidades.

User State Migration Tool

O User State Migration Tool do Windows (USMT) migra todos os arquivos e configurações do usuário para o novo sistema operacional, como mostrado na Figura 1. Baixe e instale o USMT da Microsoft. Use o software armazenar os arquivos e configurações do usuário que estão salvos em um local diferente do OS. Depois que o novo sistema operacional estiver instalado, baixe e instale USMT novamente para carregar os arquivos e configurações do usuário no novo OS.

Transferência Fácil do Windows

Se um usuário estiver trocando de um computador antigo para um novo, você pode usar a Transferência Fácil do Windows para migrar arquivos e configurações pessoais, como mostrado na Figura 2. Você pode realizar a transferência de arquivos através de um cabo USB, CD ou DVD, unidade flash USB, disco rígido externo ou via rede.

Depois de executar a Transferência Fácil do Windows, você pode verificar o registro dos arquivos transferidos. Para acessar a Transferência Fácil do Windows no Windows 7 ou Windows Vista, use o seguinte caminho:

Iniciar> Todos os Programas> Acessórios> Ferramentas do Sistema> Transferência Fácil do Windows

No Windows XP, você deve primeiro fazer o download do programa Transferência Fácil do Windows. Depois de baixar e instalar, acesse-o usando o seguinte caminho:

Iniciar> Todos os Programas> Transferência Fácil do Windows

A Transferência Fácil do Windows substituiu o Assistente de Transferência de Arquivos e Configurações do Windows XP.

Assistente de Transferência de Arquivos e Configurações do Windows XP

O Assistente de Transferência de Arquivos e Configurações do Windows XP, como mostrado na Figura 3, transfere arquivos e configurações de um computador antigo para um novo. Ele permite que o usuário selecione quais arquivos e configurações serão transferidos. Você pode transferir arquivos usando mídias de armazenamento, uma conexão por cabo ou conexão de rede. Para acessar o Assistente de Transferência de Arquivos e Configurações, use o seguinte caminho:

Iniciar> Todos os Programas> Acessórios> Ferramentas do Sistema> Assistente de Transferência de Arquivos e Configurações
