Interface Gráfica do Windows e Painel de Controle

Ferramentas Administrativas

A console de serviços, conforme apresentado na figura, permite gerenciar todos os serviços em seu computador e também em computadores remotos. Um serviço é um tipo de aplicação que roda em segundo plano, realizando um objetivo específico ou aguardando por uma solicitação. Para reduzir os riscos de segurança, apenas inicie os serviços necessários. Você pode usar as seguintes configurações ou estados para controlar um serviço:

  • Automático - O serviço começa assim que o computador é iniciado. Este modo prioriza os serviços mais importantes para iniciarem imediatamente quando o sistema operacional for iniciado.

  • Automático (Atraso na Inicialização) - O serviço começa após a inicialização dos serviços configurados no modo Automático. O modo Automático (Atraso na Inicialização) está disponível apenas no Windows 7 e Windows Vista.

  • Manual - O serviço deve ser iniciado manualmente.

  • Desativado - O serviço não pode ser iniciado até que ele seja ativado.

  • Parado - O serviço não está sendo executado.

Para abrir a console de serviços, use o seguinte caminho:

Iniciar > Painel de Controle > Ferramentas Administrativas > Serviços

Para visualizar o console de serviços em um computador remoto, siga estes passos:

Passo 1. Na árvore de console, clique com o botão direito do mouse em Serviços (Local) e selecione Conectar a outro computador.

Passo 2. Na janela exibida, digite o nome do computador ou navegue para encontrar o dispositivo que deseja gerenciar.
