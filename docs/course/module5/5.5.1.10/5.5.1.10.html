Plano de Manutenção Preventiva para o Sistema Operacional

Às vezes, a instalação de um aplicativo ou driver de hardware pode causar instabilidade ou criar problemas inesperados. Desinstalar o aplicativo ou driver de hardware geralmente corrige o problema. Se a desinstalação não resolver o problema, restaure o computador para um momento anterior, utilizando o utilitário Restauração do Sistema.

Os pontos de restauração contêm informações sobre o sistema e suas configurações de registro. Se um computador travar ou uma atualização causar problema, o computador poderá ser revertido para uma configuração anterior usando um ponto de restauração. A restauração do sistema não faz backup de arquivos de dados pessoais e nem recupera os arquivos pessoais que foram corrompidos ou apagados. Sempre use um sistema de backup dedicado, como uma unidade de fita, disco óptico ou dispositivo de armazenamento USB.

Nas seguintes situações, um técnico deve sempre criar um ponto de restauração antes de fazer alterações ao sistema:

  • Ao atualizar o sistema operacional

  • Ao instalar ou atualizar um hardware

  • Ao instalar um aplicativo

  • Ao instalar um driver

Para abrir o utilitário Restauração do Sistema e criar um ponto de restauração, use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Ferramentas do sistema > Restauração do sistema
