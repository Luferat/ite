Instalação do Sistema Operacional

Estrutura de Diretório e Atributos de Arquivos

Locais de Arquivos do Usuário

Por padrão, o Windows 7 e Windows Vista armazenam a maioria dos arquivos criados pelos usuários na pasta C:\Usuários\nome_do_usuário\. O Windows XP usa a pasta C:\Documents and Settings\nome_do_usuário\.

A pasta de cada usuário contém pastas de músicas, vídeos, sites, fotos, entre outros, conforme apresentado na Figura 1. Muitos programas também armazenam dados de usuário específicos nesta pasta. Se um único computador tiver muitos usuários, cada usuário terá sua própria pasta que contém os seus favoritos, itens do desktop e cookies. Os cookies são arquivos que contêm informações das páginas da web que o usuário visitou.

Pasta do Sistema

Quando o sistema operacional Windows é instalado, todos os arquivos que são usados ​​para executar o computador estão localizados na pasta C:\Windows\system32. O conteúdo da pasta de Sistema é mostrado na Figura 2.

Fontes

A pasta C:\Windows\Fonts contém as fontes instaladas no computador. As fontes vêm em vários formatos, incluindo TrueType, OpenType, Composite e PostScript. Alguns exemplos de tipos de fontes são Arial, Times New Roman e Courier. Você pode acessar a pasta Fontes através do Painel de Controle. Você pode instalar as fontes usando o menu Arquivo > Instalar Nova Fonte.

Arquivos Temporários

A pasta de arquivos temporários contém arquivos criados pelo sistema operacional e programas que são necessários por um curto período de tempo. Por exemplo, os arquivos temporários podem ser criados enquanto um aplicativo está sendo instalado para disponibilizar mais memória RAM para outros programas.

Quase todos os programas utilizam arquivos temporários, que normalmente são excluídos automaticamente quando o aplicativo ou o sistema operacional é finalizado. No entanto, alguns arquivos temporários devem ser apagados manualmente. Como os arquivos temporários ocupam espaço no disco rígido que poderia ser usado para outros arquivos, é uma boa ideia excluí-los a cada dois ou três meses.

No Windows Vista e Windows 7, os arquivos temporários geralmente estão localizados nas seguintes pastas:

  • C:\Windows\Temp

  • C:\Usuário\Nome_do_Usuário\AppData\Local\Temp

  • %USERPROFILE%\AppData\Local\Temp

No Windows XP, os arquivos temporários geralmente estão localizados nas seguintes pastas:

  • C:\Temp

  • C:\Tmp

  • C:\Windows\Temp

  • C:\Windows\Tmp

  • C:\Documents and Settings\%USERPROFILE%\Local Settings\Temp

NOTA: %USERPROFILE% é uma variável de ambiente definida pelo sistema operacional com o nome do usuário que está atualmente conectado ao computador. As variáveis ​​de ambiente são utilizadas pelo sistema operacional, aplicativos e programas de instalação de software.

Para ver as variáveis ​​de ambiente que estão configuradas no Windows 7, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Configurações Avançadas do Sistema > guia Avançado > Variáveis de Ambiente

Para ver as variáveis ​​de ambiente que estão configuradas no Windows Vista, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Configurações Avançadas do Sistema > guia Avançado > Variáveis de Ambiente

Para ver as variáveis ​​de ambiente que estão configuradas no Windows XP, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Avançado > Variáveis ​​de Ambiente

Arquivos de Programas

A pasta Arquivos de Programas é usada pela maioria dos programas de instalação de aplicativo para instalar o software. Em sistemas de 32 bits, os programas geralmente são instalado na pasta C:\Arquivos de Programas. Em sistemas de 64 bits, os programas de 64 bits geralmente são instalados na pasta C:\Arquivos de Programas, enquanto os programas de 32 bits geralmente são instalados na pasta C:\Arquivos de Programas (x86).
