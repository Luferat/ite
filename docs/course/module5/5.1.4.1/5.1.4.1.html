Sistemas Operacionais Modernos

Atualizações do Sistema Operacional

Um sistema operacional deve ser atualizado periodicamente para manter a compatibilidade com os hardwares e softwares mais recentes. Também é necessário atualizar um sistema operacional mesmo quando o fabricante parar de oferecer suporte. Atualizar um sistema operacional pode aumentar o desempenho. Novos produtos de hardware muitas vezes exigem a última versão do sistema operacional instalada para funcionarem corretamente. Atualizar um sistema operacional pode ser caro, mas você pode ganhar uma funcionalidade melhorada através dos novos recursos, e terá suporte para os hardwares mais recentes.

NOTA: Quando as versões mais recentes de um Sistema Operacional são liberadas, o suporte para versões mais antigas, eventualmente, é descontinuado.

Antes de atualizar o sistema operacional, verifique os requisitos mínimos de hardware do novo sistema operacional para garantir que ele pode ser instalado com êxito no computador. Verifique também o Centro de Compatibilidade do Windows do Windows 7 e Vista, ou o Windows XP HCL, para garantir que o hardware é compatível com o novo sistema operacional.

Upgrade Advisor

Microsoft oferece um utilitário gratuito chamado Upgrade Advisor, que escaneia o sistema para procurar problemas de incompatibilidade de hardware e software antes de atualizar para as versões mais recentes do sistema operacional Windows. O Upgrade Advisor gera um relatório de todos os problemas e, em seguida, guia-o em determinados passos para resolver os problemas. Você pode baixar o Upgrade Advisor no site da Microsoft.

Para usar o Upgrade Advisor no Windows 7, siga estes passos:

Passo 1. Baixe e execute o Windows 7 Upgrade Advisor no site da Microsoft.

Passo 2. Clique em Iniciar a verificação. O programa verifica o seu hardware, os dispositivos e os softwares instalados. Um relatório de compatibilidade é apresentado.

Passo 3. Clique em Salvar Relatório, se você quiser mantê-lo ou imprimi-lo mais tarde.

Passo 4. Examine o relatório. Registre quaisquer correções recomendadas para os problemas encontrados.

Passo 5. Clique em Fechar.

Depois de fazer as alterações de hardware, dispositivos ou software, a Microsoft recomenda a execução do Upgrade Advisor novamente antes de instalar o novo sistema operacional.
