Interface Gráfica do Windows e Painel de Controle

Ferramentas Administrativas

O Windows contém vários utilitários para gerenciar permissões e usuários ou configurar componentes e serviços dos computadores. A console de gerenciamento do computador, conforme apresentado na imagem, permite gerenciar muitos aspectos do seu computador e de computadores remotos.

A console de gerenciamento do computador permite o acesso a uma série de utilitários, incluindo:

  • Agendador de Tarefas

  • Visualizador de Eventos

  • Pastas Compartilhadas

  • Usuários e Grupos Locais

  • Desempenho

  • Gerenciador de Dispositivos

  • Gerenciamento de Disco

Para abrir a console de gerenciamento do computador, use o seguinte caminho:

Iniciar > Painel de Controle > Ferramentas Administrativas > Gerenciamento do Computador

Para visualizar a console de gerenciamento de um computador remoto, siga estes passos:

Passo 1. Na console, clique com o botão direito do mouse em Gerenciamento do Computador (Local) e selecione Conectar a outro computador.

Passo 2. Na janela exibida, digite o nome do computador ou navegue para encontrar o dispositivo que deseja gerenciar.
