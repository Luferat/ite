Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

O utilitário Sistema no Painel de Controle do Windows permite que todos os usuários visualizem informações básicas do sistema, ferramentas de acesso e configurar as definições avançadas do sistema.

Para acessar o utilitário Sistema, conforme apresentado na Figura 1, utilize o seguinte caminho:

Iniciar > Painel de Controle > Sistema

Você pode acessar as diversas definições clicando nos links.

Quando um usuário clicar no link Gerenciador de Dispositivos, o utilitário Gerenciador de Dispositivos será aberto. Quando um dos outros links é aberto, o utilitário Propriedades do Sistema aparece com as seguintes guias:

  • Nome do Computador - Exibe ou modifica as configurações de nome e grupo de trabalho para um computador, bem como altera o domínio ou grupo de trabalho.

  • Hardware - Acessa o Gerenciador de Dispositivos ou ajusta as configurações de instalação do dispositivo.

  • Avançado - Define as configurações de desempenho, perfis de usuário, inicialização e recuperação.

  • Proteção do Sistema - Restauração de Sistema e configurações de proteção e segurança.

  • Remoto - Ajusta as definições para a Assistência Remota e Desktop Remoto.

Configurações de Desempenho

Para melhorar o desempenho do sistema operacional, você pode alterar algumas das configurações que o seu computador usa, tais como definições de configuração de memória virtual, conforme apresentado na Figura 2. O sistema operacional usa a memória virtual quando um computador não tem memória RAM suficiente disponível para executar um programa. Se a RAM suficiente não estiver disponível, a memória virtual move os dados da RAM e os coloca em um arquivo de paginação no disco rígido. Um arquivo de paginação é um lugar onde os dados são armazenados até que a RAM suficiente esteja disponível para processar os dados. Este processo é muito mais lento do que acessar diretamente a memória RAM. Se um computador tem uma pequena quantidade de memória RAM, considere a compra de RAM adicional para reduzir a paginação.

Para exibir a configuração de memória virtual no Windows 7, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Configurações avançadas do sistema > Desempenho > Configurações > Avançado > Alterar

No Windows Vista, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Configurações avançadas do sistema > Continuar > guia Avançado > Área de Performance > Configurações > Avançado > Alterar

No Windows XP, utilize o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Avançado > Performance > Configurações > guia Avançado > Alterar

Windows ReadyBoost

Se um usuário não é capaz de instalar mais RAM, eles podem usar um dispositivo flash externo e o Windows ReadyBoost, a fim de melhorar o desempenho do Windows 7 e Windows Vista. O Windows ReadyBoost permite que o sistema operacional trate um dispositivo flash externo, exemplo: pen drive USB, como cache de disco rígido quando não há memória RAM suficiente disponível. Para ativar o Windows ReadyBoost, um usuário deve inserir um dispositivo flash e usar o seguinte caminho:

Iniciar > Computador > botão direito do mouse no dispositivo flash externo desejado > selecione Propriedades > Clique na guia ReadyBoost

Uma vez que o ReadyBoost foi ativado para o dispositivo desejado, o usuário deve determinar a quantidade de espaço no dispositivo que será reservada como cache. Um mínimo de 256 MB devem ser selecionados, com um máximo de 4 GB para o sistema de arquivo FAT32 e 32GB no sistema de arquivo NTFS.
