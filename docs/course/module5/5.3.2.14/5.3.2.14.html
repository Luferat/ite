Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

O utilitário Opções de Energia do Windows permite que você reduza o consumo de energia de determinados dispositivos, ou de todo o sistema, como mostrado na figura. Utilize as opções de energia para maximizar o desempenho ou economizar energia, configurando o plano de energia do sistema. Os planos de energia são um conjunto de configurações de hardware e do sistema que controlam o uso de energia do computador. No Windows XP, estes planos são chamados de Esquemas de Energia.

Ambos Windows 7 e Windows Vista têm planos de energia pré-definidos, enquanto o Windows XP tem esquemas de energia pré-definidos. Estas são as configurações padrão, criadas quando o sistema operacional é instalado. Você pode usar as configurações padrão ou os planos de energia personalizados, que são criados baseados em exigências de trabalho específicas.

NOTA: As opções de energia detectam automaticamente alguns dispositivos conectados ao computador. Portanto, as janelas de Opções de Energia variam de acordo com o hardware detectado.

Para acessar o utilitário Opções de Energia, use o seguinte caminho:

Iniciar> Painel de Controle> Opções de Energia

Você pode escolher entre as seguintes opções:

  • Exigir senha ao despertar

  • Escolher função dos botões de energia

  • Escolher a função do fechamento da tampa (apenas para laptops)

  • Criar um plano de energia

  • Escolher tempo para desligar o vídeo

  • Alterar quando o computador é suspenso

Selecionar Escolher a função dos botões de energia ou Escolher a função do fechamento da tampa define como um computador agirá quando o botão de energia ou suspender for pressionado ou quando a tampa de um laptop for fechada. Caso os usuários não queiram desligar completamente um computador, as seguintes opções estão disponíveis:

  • Não fazer nada - O computador continua a funcionar a plena potência.

  • Suspender - Documentos e aplicativos são salvos na memória RAM, permitindo que o computador ligue rapidamente. No Windows XP, essa opção é chamada de Espera.

  • Hibernar - Documentos e aplicativos são salvos em um arquivo temporário no disco rígido. Neste modo, o computador leva um pouco mais de tempo para ligar, comparado à opção Suspender.
