Ferramentas de Linha de Comando

Ao solucionar problemas no sistema operacional, pode ser necessário utilizar comandos da CLI e outras opções para executar algumas tarefas. Para acessar a CLI, utilize o seguinte caminho:

Iniciar > digite cmd na caixa de pesquisa > pressione Enter

Assim que a janela de comando estiver aberta, você poderá digitar os comandos para executar funções específicas. A figura descreve os comandos mais comuns, como usá-los e o que eles fazem.

Caso o uso de algum desses comandos seja negado, você precisará acessar a CLI como administrador. Para acessar a CLI como administrador, utilize o seguinte caminho:

Iniciar > digite cmd na caixa de pesquisa > clique com o botão direito do mouse em cmd.exe > ​​clique em Executar como administrador > clique em Sim
