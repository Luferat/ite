Estrutura de diretório e atributos de arquivo

Os arquivos de uma estrutura de diretórios aderem a uma convenção de nomenclatura Windows:

  • É permitido no máximo 255 caracteres.

  • Caracteres como barra ou barra invertida (/ \) não são permitidos.

  • Uma extensão de três ou quatro letras é adicionado ao nome do arquivo para identificar o tipo do mesmo.

  • Nomes de arquivos não são case sensitive.

Por padrão, as extensões de arquivos são ocultadas. Para exibir as extensões dos arquivos, você deve desabilitar a opção "Ocultar as extensões" no utilitário de configuração do painel de controle em Opções de Pasta, conforme apresentado na Imagem.

Para exibir as extensões de arquivo, use o seguinte caminho:

Iniciar > Painel de Controle > Opções de Pasta > Modo de Exibição > desmarque a opção Ocultar as extensões dos tipos de arquivo conhecidos

As seguintes extensões de arquivos são utilizadas:

  • .docx - Microsoft Word

  • .txt - apenas texto ASCII

  • .jpg - formato gráfico

  • .pptx - Microsoft PowerPoint

  • .zip - formato de compressão

A estrutura de diretório mantém um conjunto de atributos para cada arquivo que controla a forma de como o mesmo pode ser visualizado ou alterado. Estes são os atributos de arquivos mais comuns:

  • R - O arquivo é somente leitura.

  • A - O arquivo será arquivado na próxima vez que for realizado um backup de disco.

  • S - O arquivo é marcado como um arquivo de sistema, e um aviso é dado se for feita alguma tentativa de excluir ou modificar o mesmo.

  • H - O arquivo está oculto.
