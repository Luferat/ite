Modos de Inicialização

O Windows pode iniciar em muitos modos diferentes. Pressionar a tecla F8 durante o processo de inicialização abrirá o menu Opções de inicialização avançadas do Windows, conforme apresentado na figura. Isso permite que os usuários selecionem como o Windows será inicializado. As seguintes opções de inicialização são encontradas com frequência:

  • Modo de Segurança - Inicia o Windows, porém carrega apenas os drivers dos componentes básicos, como o teclado e o monitor.

  • Modo de Segurança com Rede - Inicia o Windows semelhante ao modo de segurança, porém também carrega os drivers dos componentes de rede.

  • Modo de Segurança com Prompt de Comando - Inicia o Windows e carrega o prompt de comando em vez da GUI.

  • Última Configuração Válida - Carrega as configurações que foram usadas na última vez que o Windows foi iniciado com êxito. O Windows faz isso ao acessar uma cópia do registro que é criado para está finalidade.

NOTA: A Última Configuração Válida não é útil a menos que seja aplicada imediatamente após ocorrer uma falha. Se a máquina for reiniciada e conseguir carregar o Windows, o registro será atualizado com a informação defeituosa.
