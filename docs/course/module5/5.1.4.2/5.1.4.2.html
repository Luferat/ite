Atualização do Sistema Operacional

O processo de atualização do sistema operacional de um computador pode ser mais rápido do que realizar uma nova instalação. O processo de atualização varia de acordo com a versão. Por exemplo, o utilitário de configuração do Windows 7 substitui arquivos existentes do Windows Vista por arquivos do Windows 7. No entanto, os programas e configurações existentes são salvos.

A versão de um sistema operacional determina as opções de atualização disponíveis. Por exemplo, um S.O. de 32 bits não pode ser atualizado para um S.O. de 64 bits. Outro exemplo é que o Windows XP não pode ser atualizado para o Windows 7. Antes de tentar uma atualização, verifique no site do desenvolvedor do sistema operacional uma lista dos possíveis caminhos de atualização.

NOTA: Antes de executar uma atualização, faça um backup de todos os dados no caso de houver um problema com a instalação.

Para atualizar o sistema operacional para Windows 7, siga estes passos:

Passo 1. Insira o disco do Windows 7 na unidade óptica. Uma janela de configuração aparecerá.

Passo 2. Selecione a opção Instalar agora.

Passo 3. Será solicitado que você baixe as atualizações importantes para a instalação.

Passo 4. Leia e concorde com o Contrato de Licença de Usuário Final (EULA) e clique em Avançar.

Passo 5. Clique em Atualizar. O sistema começa a copiar os arquivos de instalação.

Passo 6. Siga as instruções para concluir a atualização. Quando a instalação estiver concluída, o computador será reiniciado.

NOTA: Antes de atualizar do Windows XP para o Windows Vista, você deve instalar o Windows XP Service Pack 2 ou 3.

Para atualizar o sistema operacional para o Windows Vista, siga estes passos:

Passo 1. Insira o disco do Windows Vista na unidade óptica. Uma janela de configuração aparecerá.

Passo 2. Selecione Instalar o Windows Vista.

Passo 3. Será solicitado que você baixe as atualizações importantes.

Passo 4. Digite a chave do produto e, em seguida, aceite o Contrato de Licença de Usuário Final (EULA).

Passo 5. Clique em Atualizar. O sistema começa a copiar os arquivos de instalação.

Passo 6. Siga as instruções para concluir a atualização. Quando a instalação estiver concluída, o computador será reiniciado.

Para atualizar o sistema operacional para o Windows XP, siga estes passos:

Passo 1. Insira o CD do Windows XP na unidade óptica.

Passo 2. Selecione Iniciar > Executar. (O Assistente de Instalação deve ser iniciado automaticamente quando o disco é inserido na unidade óptica.)

Passo 3. Na caixa Executar, onde D é a unidade para a unidade óptica, digite D:\i386\winnt32 e pressione Enter. A janela Bem-vindo ao Assistente de Instalação do Windows XP será exibida.

Passo 4. Escolha Atualizar para o Windows XP e clique em Avançar. A página do Contrato de Licença será exibida.

Passo 5. Leia o contrato de licença e clique no botão para aceitá-lo.

Passo 6. Clique em Próximo. A página Atualização para o Sistema de Arquivos NTFS do Windows XP será exibida.

Passo 7. Siga as instruções para concluir a atualização. Quando a instalação estiver concluída, o computador será reiniciado.
