Interface Gráfica do Windows e Painel de Controle

Acessórios

Os técnicos podem usar a Conexão de Área de Trabalho Remota e a Assistência Remota do Windows para reparar e atualizar os computadores. A Conexão de Área de Trabalho Remota, conforme apresentado na Figura 1, permite que os técnicos visualizem e controlem um computador remotamente. A Assistência Remota do Windows, conforme apresentado na Figura 2, permite que os técnicos ajudem seus clientes a resolver problemas remotamente. A Assistência Remota do Windows também permite ao cliente visualizar o que está sendo reparado ou atualizado em seu computador.

Para acessar a Conexão de Área de Trabalho Remota no Windows 7 ou no Windows Vista, use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Conexão de Área de Trabalho Remota

Para o Windows XP, a Conexão de Área de Trabalho Remota está disponível apenas na versão Windows XP Professional. Para acessar a Conexão de Área de Trabalho Remota no Windows XP Professional, use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Comunicações > Conexão de Área de Trabalho Remota

Antes da Assistência Remota do Windows puder ser utilizada, ela deve ser ativada. Para ativar e acessar a Assistência Remota do Windows, siga estes passos:

Passo 1. Clique em Iniciar > botão direito em Computador > Propriedades.

Passo 2. Dentro da janela Sistema, clique no link Configurações remotas.

Passo 3. Verifique se a check box "Permitir conexões de Assistência Remota para este Computador" está marcada.

Passo 4. Clique em Aplicar > OK.

Para acessar a Assistência Remota do Windows no Windows 7 ou no Vista, use o seguinte caminho:

Iniciar > Todos os Programas > Manutenção > Assistência Remota do Windows

Para acessar a Assistência Remota no Windows XP, use o seguinte caminho:

Iniciar > Todos os Programas > Assistência Remota
