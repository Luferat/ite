Interface Gráfica do Windows e Painel de Controle

Ferramentas de Linha de Comando

O utilitário Executar, mostrado na Figura 1, permite que você digite os comandos para configurar as definições do Windows. Muitos destes comandos são usados ​​para diagnóstico e modificações do sistema.

Para acessar o Executar do Windows 7, use o seguinte caminho:

Iníciar > caixa de pesquisa > digite Executar > Enter

Para acessar o utilitário Executar no Windows Vista, use o seguinte caminho:

Iniciar > Iniciar Pesquisa > digite Executar > Enter

Para acessar o utilitário Executar no Windows XP, use o seguinte caminho:

Iniciar > Executar

Esta é uma lista de comandos comuns:

  • CMD - Usado para abrir o prompt de comando, conforme apresentado na Figura 2.

  • DXDIAG - Exibe detalhes de todos os componentes e drivers DirectX que estão instalados em um computador, conforme apresentado na Figura 3. Use este utilitário para garantir que o DirectX está instalado e configurado corretamente.

  • EXPLORER - Abre o Windows Explorer.

  • MMC - Abre a Console de Gerenciamento Microsoft (MMC), mostrado na Figura 4, que permite que você organize ferramentas de gestão, chamado de snap-ins, em um único local, facilitando a administração. Você também pode adicionar links de páginas web, tarefas, controles ActiveX e pastas. Você pode criar quantas MMCs personalizadas forem necessárias, cada uma com um nome diferente. Isto é útil quando vários administradores gerenciaram diferentes aspectos do mesmo computador. Cada administrador pode ter uma MMC individualizada para monitorar e configurar o computador. Você deve ter privilégios administrativos para acessar a MMC.

  • MSCONFIG - Abre o utilitário de Configuração do Sistema, conforme apresentado na Figura 5, que executa procedimentos de diagnóstico nos arquivos de inicialização do Windows. Você deve estar logado com permissões de administrador para completar o processo de solução de problemas. Use o MSCONFIG quando o computador for inicializado e não carregar corretamente o Windows.

  • MSINFO32 - Apresenta um resumo completo do sistema do computador, incluindo componentes de hardware e informação de softwares, tal como mostrado na Figura 6.

  • MSTSC - Abre a Conexão de Área de Trabalho Remota

  • NOTEPAD - Abre o Bloco de Notas, conforme apresentado na Figura 7, que é um editor de texto básico.

  • REGEDIT - Abre o utilitário Editor do Registro, conforme apresentado na Figura 8, que permite ao usuário editar o registro. Usar o Editor do Registro incorretamente pode causar problemas de hardware, aplicativo ou no sistema operacional, incluindo problemas que exigem a reinstalação do S.O.

  • SERVICES.MSC - Abre o utilitário Serviços.

MSCONFIG

MSCONFIG é oficialmente chamado de Configuração do Sistema no Windows 7 e Windows Vista, e é chamado Utilitário de Configuração do Sistema Microsoft no Windows XP. Este utilitário pode ser usado para desativar ou reativar unidades do dispositivo, serviços do Windows e softwares. Ele também pode alterar parâmetros de inicialização.

As cinco guias a seguir são usadas ​​para gerenciar as diferentes características:

  • Geral - Utilizado para selecionar os métodos Inicialização Normal, Diagnóstico e Seletiva

  • Inicialização do Sistema - Oferece opções de inicialização que podem ser habilitadas e desabilitadas.

  • Serviços - Usado para desativar e ativar serviços em execução

  • Inicialização de Programas - Usado para ativar ou desativar os itens que são carregados automaticamente quando o computador é ligado

  • Ferramentas - Usado para executar os recursos do Windows que normalmente são encontrados no Painel de Controle
