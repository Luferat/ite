Virtualização Client-Side

Propósito e Requisitos da Virtualização

O software que cria e gerencia uma máquina virtual em uma máquina host é chamado de hypervisor ou Virtual Machine Manager (VMM). O hypervisor pode executar várias máquinas virtuais em um único computador hospedeiro. Cada máquina virtual executa seu próprio sistema operacional. O número de máquinas virtuais que podem ser executadas depende dos recursos de hardware da máquina hospedeira. O hypervisor aloca recursos físicos do sistema, como CPU, RAM e disco rígido para cada máquina virtual, de acordo com a sua necessidade. Isto garante que a operação de uma máquina virtual não interfira no funcionamento de uma outra.

Existem dois tipos de hypervisors: Tipo 1 (native) e Tipo 2 (hosted). O hypervisor tipo 1 é executado diretamente no hardware de um host, gerenciando assim a alocação de recursos do sistema para os sistemas operacionais convidados. O tipo 2 é hospedado por um OS. O Windows Virtual PC é um exemplo de hypervisor tipo 2.

Windows Virtual PC

Windows Virtual PC é a plataforma de virtualização para o Windows 7. O Virtual PC permite particionar recursos do sistema para máquinas virtuais que executam um sistema operacional Windows, utilizando um cópia licenciada do Windows 7, Windows Vista e Windows XP. Você pode baixar o Virtual PC a partir do site da Microsoft. A figura mostra os requisitos de hardware necessários para executar o Virtual PC. É necessário utilizar o Virtual PC para executar o programa Windows XP Mode no Windows 7.

Windows XP Mode

O Windows XP Mode é um programa disponível para as versões Professional, Enterprise e Ultimate do Windows 7. O Windows XP Mode usa tecnologia de virtualização que permite aos usuários executar programas do Windows XP no Windows 7. Ele abre uma máquina virtual no área de trabalho do Windows 7, fornecendo uma versão totalmente funcional do Windows XP, incluindo o acesso a todos os recursos do sistema. Depois de instalar um programa no Windows XP Mode, você poderá executar o programa no modo XP e acessá-lo a partir do menu Iniciar do Windows 7.

NOTA: Antes de usar o Windows XP Mode, baixe e instale o Windows Virtual PC.

Para acessar o Windows XP Mode no Windows 7, siga estes passos:

Passo 1. Clique em Iniciar> Todos os Programas.

Passo 2. Clique em Windows Virtual PC> Windows XP Mode.
