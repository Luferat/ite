Interface Gráfica do Windows e Painel de Controle

Desktop, Ferramentas e Aplicações

Depois que o S.O. foi instalado, você pode personalizar a área de trabalho do computador para atender as suas necessidades individuais. A área de trabalho do computador é uma representação gráfica do espaço de trabalho, comumente chamado de GUI. A área de trabalho possui ícones, barras de ferramentas e menus para manipular arquivos. Você pode adicionar ou alterar imagens, sons e cores para proporcionar um visual mais personalizado. Juntos, esses itens personalizáveis ​​compõem um tema. O Windows 7 e Windows Vista usam um tema padrão chamado Aero. O Aero tem bordas de janelas translúcidas, numerosas animações e ícones que são imagens em miniatura do conteúdo de um arquivo. Devido aos gráficos avançados necessários para executar o tema, o Aero está disponível apenas em computadores que atendam aos seguintes requisitos de hardware:

  • Processador de 1 GHz 32 ou 64 bits

  • 1 GB de RAM

  • Placa de vídeo de 128 MB

  • Processador gráfico classe DirectX9 que suporte o Modelo de Driver de Exibição do Windows, Pixel Shader 2.0 em hardware e 32 bits por pixel.

NOTA: O Windows 7 Starter e Windows Vista Home Basic não incluem o tema Aero.

O Windows 7 inclui os seguintes novos recursos do Aero:

  • Shake - Minimiza todas as janelas que não estão sendo utilizadas, clicando na barra de título de uma janela e agitando o mouse. Para maximizar as janelas novamente, clique na barra de título da janela visível e agite o mouse.

  • Peek - Visualiza os ícones e gadgets na área de trabalho apontando o mouse na borda direita da barra de tarefas para tornar todas as janelas transparentes.

  • Snap - Redimensiona uma janela arrastando-a para um dos cantos da tela. Arrastar a janela para a borda esquerda da área de trabalho encaixa a janela na metade esquerda da tela. Arrastar a janela para a borda direita da área de trabalho encaixa a janela na metade direita da tela. Arrastar a janela para a borda superior da área de trabalho maximiza a janela para preencher a tela inteira.

Gadgets

No Windows 7 e Windows Vista, os usuários podem colocar Gadgets na área de trabalho. Gadgets são pequenos aplicativos, como jogos, notas ou um relógio. Milhares de Gadgets estão disponíveis para acessar diferentes tipos de dados. A Figura 1 mostra os Gadgets de tempo, calendário e relógio em um desktop do Windows 7. Você pode alterar a posição dos gadgets para os lados e cantos da área de trabalho, bem como alinhá-los com outros gadgets.

NOTA: O Windows XP não possui Gadgets.

Para adicionar gadgets no desktop, siga estes passos:

Passo 1. Clique com o botão direito do mouse em qualquer lugar da área de trabalho e selecione Gadgets.

Passo 2. Execute qualquer dos seguintes procedimentos:

  • Arraste e solte o gadget para o desktop.

  • Clique duas vezes no gadget para adicioná-lo à área de trabalho.

  • Clique com o botão direito do mouse no gadget e selecione Adicionar.

Passo 3. Para encaixar um gadget, arraste-o para o local de trabalho desejado. O gadget se alinha com as bordas da tela e outros gadgets.

No Windows Vista, você também pode personalizar um recurso chamado Sidebar, como mostrado na Figura 2. Sidebar é um painel gráfico na área de trabalho que mantém os gadgets organizados. O Sidebar não está disponível no Windows 7.
