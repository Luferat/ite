Técnicas de Manutenção Preventiva para os Sistemas Operacionais

Plano de Manutenção Preventiva para Sistema Operacional

Atualizações de Drivers de Dispositivos

A atualização regular dos drivers dos dispositivos deve fazer parte do seu programa de manutenção preventiva para garantir que os seus drivers são atuais. De tempos em tempos, os fabricantes lançam novos drivers para resolver problemas dos drivers atuais. Verifique se há atualizações de drivers quando seu hardware não funcionar corretamente ou para evitar problemas futuros. Também é importante atualizar os drivers que corrigem problemas de segurança. Se uma atualização de driver não funcionar corretamente, utilize o recurso Reverter Driver para reverter o mesmo para o driver instalado anteriormente.

Atualizações do Sistema Operacional

A Microsoft libera atualizações para tratar de questões de segurança e outros problemas de funcionalidade. Você pode instalar as atualizações individuais manualmente, através do site da Microsoft, ou automaticamente, usando o utilitário Atualizações Automáticas do Windows. Downloads que contêm várias atualizações são chamados service packs. Um service pack geralmente contém todas as atualizações de um sistema operacional. Instalar de um service pack é uma boa maneira de fazer com que o seu sistema operacional seja atualizado rapidamente. Crie um ponto de restauração e faça backup dos dados críticos antes de instalar um service pack. Adicione as atualizações do sistema operacional no seu programa de manutenção preventiva para garantir que o seu sistema tenha as mais recentes funcionalidades e correções de segurança.

Atualizações de Firmware

Atualizações de firmware são menos frequentes e comuns do que as atualizações de drivers. Os fabricantes lançam novas atualizações de firmware para resolver problemas que não podem ser corrigidos através de atualizações de drivers. As atualizações de firmware podem aumentar a velocidade de certos tipos de hardware, habilitar novos recursos ou aumentar a estabilidade de um produto. Siga, com cuidado, as instruções do fabricante ao realizar uma atualização de firmware para evitar que o hardware se torne inutilizável. Faça uma pesquisa completa da atualização antes de instalá-la, pois pode não ser possível reverter para o firmware original. Verificar se há atualizações de firmware deve fazer parte do seu programa de manutenção preventiva.
