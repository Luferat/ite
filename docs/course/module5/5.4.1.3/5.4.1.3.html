Propósito e Requisitos da Virtualização

Todas as máquinas virtuais exigem que os requisitos básicos do sistema sejam atendidos, como uma quantidade mínima de espaço em disco rígido ou memória RAM. Os requisitos mínimos de sistema necessários para utilizar o Windows Virtual PC são apresentados na figura.

Tal como os computadores físicos, as máquinas virtuais são suscetíveis a ameaças e ataques maliciosos. Os usuários devem instalar o software de segurança, habilitar o Firewall do Windows e atualizar os patches e drivers.

Para se conectar à Internet, uma máquina virtual utiliza um adaptador de rede virtual. O adaptador de rede virtual funciona como um adaptador real em um computador físico, a não ser que ele se conecte através de um adaptador físico no host para estabelecer uma conexão com a Internet.
