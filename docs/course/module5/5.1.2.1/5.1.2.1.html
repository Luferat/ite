Sistemas Operacionais Modernos

Tipos de Sistemas Operacionais

Pode ser solicitado a um técnico escolher e instalar um sistema operacional para um cliente. Existem dois tipos diferentes de sistemas operacionais: desktop e de rede. Um sistema operacional desktop é destinado ao uso em um pequeno escritório, home office (SOHO), um ambiente com um número limitado de usuários. Um sistema operacional de rede (NOS) é projetado para um ambiente corporativo, servindo vários usuários com uma vasta gama de necessidades.

Um sistema operacional desktop tem as seguintes características:

  • Suporta um único usuário

  • Executa aplicativos de um único usuário

  • Compartilha arquivos e pastas em uma pequena rede com segurança limitada

No mercado de software atual, os sistemas operacionais de desktop mais utilizados se dividem em três grupos: Microsoft Windows, Apple Mac OS e Linux. Este capítulo se concentra em sistemas operacionais da Microsoft.

Microsoft Windows

O Windows é um dos sistemas operacionais mais populares hoje em dia. As seguintes versões do Windows estão disponíveis:

  • Windows 7 Starter - Usado em computadores e netbooks que facilitam a utilização de redes

  • Windows 7 Home Premium - usado em computadores domésticos para compartilhar mídia facilmente

  • Windows 7 Professional - usados ​​em computadores de pequenas empresas para proteger informações críticas e fazer tarefas rotineiras mais fáceis de completar

  • Windows 7 Enterprise - Usado em computadores para fornecer aumento de produtividade, segurança e recursos de gerenciamento.

  • Windows 7 Ultimate - usado em computadores para combinar a facilidade de uso do Windows 7 Home Premium com os recursos comerciais do Windows 7 Professional e proporcionar maior segurança de dados

  • Windows Vista Home Basic - usado em computadores domésticos para computação básica

  • Windows Vista Home Premium - usado em computadores domésticos para expandir a produtividade pessoal e entretenimento digital para além do básico

  • Windows Vista Business - usados ​​em computadores de pequenas empresas para aumentar a segurança e tecnologia avançada mobilidade

  • Windows Vista Enterprise - Usado em computadores para negócios que fornece o aumento de produtividade, segurança e recursos de gerenciamento

  • Windows Vista Ultimate - usado em computadores para combinar todas as necessidades dos usuários domésticos e empresariais

  • Windows XP Professional - Utilizado na maioria dos computadores que se conectam a um servidor Windows em uma rede

  • Windows XP Home - usado em computadores domésticos com segurança limitada

  • Windows XP Media Center - usados ​​em computadores de entretenimento para ver filmes e ouvir músicas

  • Windows XP 64-bit Professional - Usado para computadores com processadores de 64 bits

Apple Mac OS

Computadores da Apple são proprietários e usam um sistema operacional chamado Mac OS. O Mac OS é projetado para ser um sistema operacional com uma interface gráfica(GUI) "amigável". As versões atuais do Mac OS são baseadas em uma versão personalizada do UNIX.

Linux

Linux é baseado em UNIX, que foi introduzido na década de 1960 e é um dos sistemas operacionais mais antigos. Linus Torvalds Linux projetado em 1991 como um sistema operacional open-source(código fonte aberto). Programas de código fonte aberto(open-source) permitem que o código fonte seja distribuído e modificado por qualquer pessoa como um download gratuito ou por desenvolvedores, a um custo muito menor do que outros sistemas operacionais.
