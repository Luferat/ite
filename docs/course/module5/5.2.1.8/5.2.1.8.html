Instalação do Sistema Operacional

Procedimentos de Configuração de Disco Rígido

Quando os usuários tentam efetuar login em um dispositivo ou acessar recursos do sistema, o Windows usa o processo de autenticação para verificar se os usuários são quem eles dizem que são. A autenticação ocorre quando o usuário digitar um username e senha para acessar uma conta. Os sistemas operacionais Windows usam a autenticação Single-Sign On (SSO), que permite que os usuários façam login uma vez para acessar todos os recursos do sistema ao invés de obrigá-los a se autenticar toda cada vez que precisam acessar um recurso individual.

As contas de usuário permitem que múltiplos usuários compartilhem um único computador, cada usuário tendo seus próprios arquivos e configurações. Windows 7 e Windows Vista possuem três tipos de contas de usuário: Administrador, Padrão e Convidados. Cada tipo de conta fornece um usuário com um nível diferente de controle sobre os recursos do sistema.

Uma conta com privilégios de administrador deve ser criada quando o Windows 7 é instalado, conforme apresentado na figura. Um usuário com privilégios de administrador podem fazer alterações que afetam todos os usuários do computador, como alterar as configurações de segurança ou a instalação de um software para todos os usuários. Contas com privilégios de administrador só devem ser usadas para gerenciar um computador e não para uso regular, pois mudanças drásticas que afetam a todos podem ser feitas quando se utiliza a conta de administrador. Os invasores também procuram uma conta de administrador, pois ela é uma conta poderosa. Por esta razão, recomenda-se que uma conta de utilizador padrão seja criado para o uso regular.

As contas de usuário padrão podem ser criadas a qualquer momento. Uma conta de usuário padrão tem menos permissões do que uma conta de administrador. Por exemplo, os usuários podem ter o direito de só ler, sem poderem modificar um arquivo.

Indivíduos que não possuem uma conta de usuário padrão no computador podem usar a conta de convidado. A conta de convidado possui permissões limitadas e deve ser ativada por um administrador.

Para criar ou remover uma conta de usuário no Windows 7 e Windows Vista, use o seguinte caminho:

Iniciar > Painel de Controle > Contas de Usuário > Adicionar ou remover contas de usuário

O Windows XP possui um quarto grupo de usuários chamado Usuários Avançados (Power Users). Os Usuários Avançados têm privilégios que se estendem além dos de usuários padrão, proporcionando algumas das capacidades dadas a uma contas de administrador. Usuários avançados não são capazes de administrar plenamente os recursos do sistema. Esse grupo não foi incluído no Windows 7 ou Windows Vista.

Para criar ou remover uma conta de usuário no Windows XP, use o seguinte caminho:

Iniciar > Painel de Controle > Contas de Usuário > Selecione a guia Usuários e clique em Adicionar
