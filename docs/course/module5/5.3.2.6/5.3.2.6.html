Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

Para acessar as Opções da Internet, use o seguinte caminho:

Iniciar > Painel de Controle > Opções da Internet

A janela Opções da Internet possui as seguintes guias:

  • Geral - Configura as definições básicas da Internet, tais como a página inicial, visualização e exclusão de históricos de navegação, ajuste e configurações de busca e personalização da aparência do navegador.

  • Segurança - Ajusta as configurações de segurança para a Internet, intranet local, sites confiáveis ​​e sites restritos. Os níveis de segurança para cada zona podem variar de baixa (segurança mínima) a alta (segurança máxima).

  • Privacidade - Define as configurações de privacidade para a Internet, gerencia os serviços de localização e habilita o bloqueador de pop-up.

  • Conteúdo - Controles de Acesso dos Pais, controle de conteúdo visualizado no computador, ajuste das configurações de AutoCompletar e configurar os feeds e Web Slices que podem ser visualizados no IE.

  • Conexões - Configura a conexão com a Internet e ajusta as configurações de rede.

  • Programas - Escolhe o navegador web padrão, permite complementos do navegador, seleciona o editor HTML para o IE e os programas utilizados para os serviços de Internet.

  • Avançado - Ajusta as configurações avançadas e redefine as configurações do Internet Explorer para o estado padrão.
