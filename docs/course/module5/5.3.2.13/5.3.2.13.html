Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

Além das configurações de segurança disponíveis na Central de Ações, você pode evitar ataques maliciosos em seu sistema utilizando o Firewall do Windows, conforme apresentado na figura. Um firewall implementa uma política de segurança seletivamente permitindo e negando o tráfego de dados para um computador. A origem do nome firewall vem dos muros de tijolos e argamassa projetados para evitar a propagação de fogo de uma parte do edifício para outras.

Você pode ajustar as configurações do firewall para redes domésticas, redes de corporativas e redes públicas. Outras mudanças podem ser feitas usando as seguintes opções:

  • Permitir um programa ou recurso pelo Firewall do Windows - Determine quais programas podem se comunicar através do Firewall do Windows.

  • Alterar configurações de notificação - Usuários com privilégios administrativos podem gerenciar as notificações do Firewall do Windows.

  • Ativar ou Desativar o Firewall do Windows - Os usuários com privilégios administrativos podem habilitar ou desabilitar o Firewall do Windows.

  • Restaurar padrões - Os usuários com privilégios administrativos podem restaurar o Firewall do Windows para as configurações padrão.

  • Configurações avançadas - Usuários com privilégios administrativos podem ajustar as configurações de segurança avançadas.

Para acessar o Firewall do Windows, no Windows 7, use o seguinte caminho:

Iniciar > Painel de Controle > Firewall do Windows
