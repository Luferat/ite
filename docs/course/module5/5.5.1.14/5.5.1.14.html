Técnicas de Manutenção Preventiva para os Sistemas Operacionais

Plano de Manutenção Preventiva para o Sistema Operacional

É importante estabelecer uma estratégia de backup que inclua a recuperação de dados pessoais. Você pode usar o Utilitário de Backup da Microsoft, apresentado na figura, para realizar backups, conforme necessário. O uso do computador, bem como os requisitos organizacionais, determinam a frequência com que os dados devem ser copiados e o tipo de backup executado.

Você pode escolher entre diversos tipos de backups, dependendo das suas necessidades.

Backup Normal

Um backup normal também é chamado de backup completo. Durante um backup normal, todos os arquivos selecionados no disco são arquivados na mídia de backup. Estes arquivos são marcados como arquivados, limpando o bit de arquivo.

Backup de Cópia

Um backup de cópia copia todos os arquivos selecionados. Ele não marca os arquivos como arquivado.

Backup Incremental

O backup incremental faz backup de todos os arquivos e pastas que foram criados ou modificados desde o último backup normal ou incremental. Ele marca os arquivos como arquivados, limpando o bit de arquivo. O efeito disso é avançar a ponto de partida dos backups diferenciais, sem ter que re-arquivar todo o conteúdo da unidade. Para recuperar um backup incremental, o último backup completo deve ser recuperado, seguido por todos os backups incrementais em ordem.

Backup Diferencial

O backup diferencial realiza o backup de todos os arquivos e pastas que foram criados ou modificados desde de o último backup normal ou incremental. Um backup diferencial não marca os arquivos como arquivado. As cópias são feitas a partir do mesmo ponto de partida de um backup incremental ou completo realizado. Fazer backups diferenciais é importante, pois apenas os últimos backups completos e diferenciais são necessários para restaurar todos os dados.

Backup Diário

Os backups diários armazenam os arquivos que foram alterados no dia do backup. Backups diários não modificam o bit de arquivo.

Ele pode demorar um tempo longo para executar o backup. Se a estratégia de backup for seguida com atenção, não é necessário fazer o backup de todos os arquivos toda vez. Em vez disso, apenas os arquivos que foram alterados desde o último backup precisam ser copiados.

Para acessar o utilitário de backup no Windows 7, use o seguinte caminho:

Iniciar > Painel de Controle > Backup e Restauração

Para acessar o utilitário de backup no Windows Vista, use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Ferramentas do Sistema > Status de Backup e Configuração

Para acessar o utilitário de backup no Windows XP Professional, use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Ferramentas do Sistema > Backup
