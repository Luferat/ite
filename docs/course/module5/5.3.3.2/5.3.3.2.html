Interface Gráfica do Windows e Painel de Controle

Ferramentas Administrativas

O Visualizador de eventos, conforme apresentado na figura, registra o histórico de eventos relacionados aos aplicativos, segurança e ao sistema. Esses arquivos de log são ferramentas valiosas para o troubleshooting, pois eles fornecem as informações necessárias para identificar um problema.

Para acessar o Visualizador de eventos, use o seguinte caminho:

Iniciar > Painel de Controle > Ferramentas Administrativas > Visualizador de Eventos

O Serviço de Componentes é uma ferramenta administrativa usada por administradores e desenvolvedores para implementar, configurar e administrar objetos e componentes COM (Component Object Model). A COM é uma forma utilizada para permitir a utilização de componentes em ambientes diferentes aos quais eles foram criados.

Para acessar os Serviços de componentes, use o seguinte caminho:

Iniciar > Painel de Controle > Ferramentas Administrativas > Serviços de Componentes

A Fontes de dados é uma ferramenta administrativa utilizada pelos administradores para gerenciar, adicionar ou remover fontes de dados usando o Open Database Connectivity (ODBC). O ODBC é uma tecnologia que os programas usam para acessar uma grande variedade de bancos de dados ou fontes de dados.

Para acessar a Fontes de dados (ODBC), use o seguinte caminho:

Iniciar > Painel de Controle > Ferramentas Administrativas > Fontes de dados (ODBC)
