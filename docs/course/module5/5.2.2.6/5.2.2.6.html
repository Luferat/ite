Instalação do Sistema Operacional

Opções de Instalação Personalizada

Quando ocorre uma falha no sistema os usuários podem utilizar as seguintes ferramentas de recuperação:

  • Opções de Recuperação do Sistema

  • Recuperação Automática do Sistema (somente para o Windows XP Professional)

  • Recuperação de Partição para as Opções de Fábrica

Opções de Recuperação do Sistema

As Opções de Recuperação do Sistema são um conjunto de ferramentas que permitem aos usuários recuperar ou restaurar o sistema operacional quando ocorre uma falha. As Opções de Recuperação do Sistema são uma parte do Ambiente de Recuperação do Windows (WinRE). O WinRE é uma plataforma de recuperação baseada no Ambiente de Pré-instalação do Windows (PE). O Windows PE é um sistema operacional básico criado para preparar um computador para a instalação do Windows e ajudar os usuários a solucionar problemas de falhas no sistema operacional quando não há um sistema operacional disponível.

O WinRE pode ser acessado pressionando e segurando a tecla F8 ao iniciar o computador. Quando a tela Opções Avançadas de Inicialização aparecer, marque a opção Reparar o computador e pressione Enter para acessar as Opções de Recuperação do Sistema. Você pode então usar as ferramentas de recuperação do sistema para reparar os erros que impedem a inicialização do sistema. As seguintes ferramentas estão disponíveis no menu Opções de Recuperação do Sistema:

  • Reparo de Inicialização - Analisa o disco rígido para encontrar problemas e corrige automaticamente os arquivos de sistema ausentes ou corrompidos que impedem a inicialização do Windows

  • Restauração do Sistema - Usa pontos de restauração para restaurar os arquivos de sistema do Windows para um ponto anterior

  • Recuperação da Imagem do Sistema - Cria uma imagem do sistema que reproduz as unidades do sistema que o Windows necessita para operar

  • Diagnostico de Memória do Windows - Examina a memória do computador para detectar falhas e diagnosticar problemas

  • Prompt de Comando - Abre uma janela do prompt de comando, onde a ferramenta Bootrec.exe pode ser utilizada para reparar e solucionar problemas de inicialização do Windows. O utilitário Bootrec.exe pode ser usado com o comando fixmbr para reparar o Master Boot Record ou o comando fixboot para escrever um novo setor de boot que é compatível com o OS. Este prompt de comando substitui a Console de Recuperação do Windows XP.

Se reparar o seu computador não aparece como uma opção, os usuários podem acessar as Opções de Recuperação do Sistema no WinRE na inicialização do computador a partir da mídia de instalação ou um disco de reparação do sistema. Um disco de reparação do sistema permite aos usuários acessarem as Opções de Recuperação do Sistema, da mesma forma que a mídia de instalação. Antes de usar o disco de reparação do sistema para iniciar um computador, ele deve ser criado.

Para criar um disco de reparação do sistema no Windows 7, conforme apresentado na Imagem 1, siga estes passos:

Passo 1. Selecione Iniciar > Painel de Controle > Backup e Restauração > Criar um disco de reparação do sistema.

Passo 2. Insira um disco vazio na unidade óptica e clique em Criar Disco.

Passo 3. Teste o disco usando-o para inicializar o computador.

Passo 4. Depois que a janela Opções de Recuperação do Sistema é exibida, selecione o sistema operacional que precisa ser restaurado e clique em Avançar. As seguintes ferramentas devem estar disponíveis:

  • Reparo de Inicialização

  • Restauração do Sistema

  • Recuperação da Imagem do Sistema

  • Diagnostico de Memória do Windows

  • Prompt de Comando

NOTA: Quando utilizar um disco de recuperação, certifique-se que ele utiliza a mesma arquitetura do sistema operacional que está sendo recuperado. Por exemplo, se o computador está executando uma versão do Windows 7 64 bits, o disco de recuperação deve usar uma arquitetura de 64 bits.

Recuperação da Imagem do Sistema

O utilitário de recuperação de imagem do sistema, conforme apresentado na Imagem 2, é uma nova opção de recuperação que está incluído em todas as versões do Windows 7. Ele permite aos usuários fazer backup do conteúdo do seu disco rígido, incluindo arquivos pessoais e configurações, se o sistema operacional precisar ser restaurado.

Para criar uma imagem do sistema no Windows 7, siga estes passos:

Passo 1. Selecione Iniciar > Painel de Controle > Backup e Restauração > Criar uma imagem do sistema.

Passo 2. Selecione um local para armazenar a imagem do sistema:

  • Em um disco rígido - Armazene a imagem do sistema em um disco rígido externo.

  • Em um ou mais DVDs - Grave a imagem do sistema em um DVD.

  • Em um local de rede - Armazene a imagem do sistema em uma pasta compartilhada na rede.

Passo 3. Clique em Avançar e confirme as seleções. A imagem do sistema é criada e armazenada no local selecionado.

Recuperação Automatizada do Sistema no Windows XP Professional

Você deve criar um Automated System Recovery (ASR) definido para utilizar na recuperação. O Assistente de ASR, conforme apresentado na Imagem 3, cria um backup da situação do sistema, serviços e componentes do sistema operacional. Ele também cria um arquivo que contém informações sobre os seus discos, o backup e como restaurar o backup.

Use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Ferramentas do Sistema > Backup > clique no link Modo Avançado > Recuperação Automatizada do Sistema

Para restaurar o sistema, pressione F2 após a inicialização do disco de instalação do Windows XP. O ASR lê e restaura os discos necessários para iniciar o computador. Após as informações básicas do disco forem restauradas, o ASR instala uma versão básica do Windows e começa a restaurar o backup criado pelo Assistente do ASR.

Recuperação de Partição para as Opções de Fábrica

Alguns computadores que possuem o Windows 7 pré-instalado de fábrica contém uma seção do disco que é inacessível para o usuário. Esta partição, chamada de partição de recuperação de fábrica, contém uma imagem da partição de boot que foi criada quando o computador foi feito. Você pode usar essa partição para restaurar o computador à sua configuração original.

Ocasionalmente, a opção para chegar a esta partição está escondida, e você deve usar uma chave especial ou combinação de teclas quando o computador é iniciado. Às vezes, a opção para restaurar a partir da partição de recuperação de fábrica está localizada na BIOS. Contate o fabricante do computador para descobrir como acessar a partição e restaurar a configuração original do computador.
