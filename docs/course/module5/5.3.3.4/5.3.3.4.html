Interface Gráfica do Windows e Painel de Controle

Ferramentas Administrativas

O Monitor de Desempenho, como mostrado na imagem, possui duas partes distintas: o Monitor do Sistema e os Logs e Alertas de Desempenho. Você deve ter privilégios administrativos para acessar a console do Monitor de Desempenho.

O Monitor do Sistema exibe informações em tempo real sobre os processadores, discos, memória e uso da rede. Use o Monitor do Sistema para exibir dados detalhados sobre os recursos que você está usando para realizar tarefas específicas ou múltiplas tarefas. Os dados mostrados podem ajudar a compreender como a carga de trabalho do computador afeta os recursos do sistema, como a CPU, memória e rede. Você pode facilmente resumir os dados de uso com histogramas, gráficos e relatórios. Os dados também podem ajudar a determinar quando uma atualização pode ser necessária.

Logs e Alertas de Desempenho permite-lhe registrar os dados de desempenho e configurar alertas. Alertas notificam-o quando um uso específico caiu ou ultrapassa um limiar. Você pode configurar os alertas para criar entradas no log de eventos, enviar uma mensagem de rede, iniciar um log de desempenho, executar um programa específico ou uma combinação destes recursos.

Para abrir o Monitor de Desempenho no Windows 7, use o seguinte caminho:

Iniciar> Painel de Controle> Ferramentas Administrativas> Monitor de Desempenho

No Windows Vista, use o seguinte caminho:

Iniciar> Painel de Controle> Ferramentas Administrativas> Monitor de Confiabilidade e Desempenho> Continuar

No Windows XP, use o seguinte caminho:

Iniciar> Painel de Controle> Ferramentas Administrativas> Performance

O Diagnóstico de Memória do Windows é uma ferramenta administrativa que verifica erros presentes na memória física instalada no computador.

Para acessar o Diagnóstico de Memória do Windows no Windows 7, use o seguinte caminho:

Iniciar> Painel de Controle> Ferramentas Administrativas> Diagnóstico de Memória do Windows

Para acessar o Diagnostico de Memória do Windows no Windows Vista, use o seguinte caminho:

Iniciar> Painel de Controle> Ferramentas Administrativas> Ferramenta de Diagnostico de Memória

O Diagnostico de Memória do Windows não está incluso no Windows XP.
