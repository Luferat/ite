Ferramentas do Sistema

Os administradores podem usar as ferramentas de Informações do Sistema, conforme apresentado na figura, para coletar e exibir informações sobre os computadores locais e remotos. As ferramentas de Informações do Sistema encontram rapidamente as informações sobre os software, drivers, configurações de hardware e componentes do computador. A equipe de suporte pode usar essas informações para diagnosticar e solucionar os problemas de um computador.

Para acessar as Ferramentas Informações do Sistema, use o seguinte caminho:

Iniciar > Todos os Programas > Acessórios > Ferramentas do Sistema > Informações do Sistema

Você também pode gerar um arquivo contendo todas as informações sobre o computador e enviá-lo para outro técnico ou help desk. Para exportar um arquivo de informações do sistema, clique em Arquivo > Exportar, digite o nome do arquivo, escolha um local e clique em Salvar.

A ferramenta de Informações do Sistema no Windows XP fornece acesso a muitos outros recursos:

  • Diagnóstico de Rede - Executa vários testes para solucionar problemas de rede.

  • Restauração do Sistema - Cria ou carrega um ponto de restauração para restaurar os arquivos e as configurações do sistema.

  • Utilitário de Verificação de Assinatura de Arquivo - Verifica os arquivos de sistema que não são assinados digitalmente.

  • Ferramenta de Diagnóstico do DirectX - Realiza relatórios contendo informações detalhadas sobre os componentes do DirectX que estão instalados em seu computador.

  • Dr Watson - Depura o Windows para ajudar a diagnosticar erros de programas.
