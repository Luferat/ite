Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

Uma conta de administrador é criada quando o sistema operacional Windows é instalado. Para criar uma conta de usuário, abra o utilitário Contas de usuário, conforme apresentado na Figura 1, e selecione Iniciar > Painel de Controle > Contas de Usuário.

O utilitário Contas de Usuário fornece opções para lhe ajudar a gerenciar sua senha, alterar sua imagem, mudar o nome e tipo de conta, gerenciar outra conta e alterar as configurações de Controle de Conta de Usuário (UAC).

NOTA: Alguns recursos do utilitário Contas de Usuário requerem privilégios administrativos e pode não ser acessíveis a uma conta de usuário padrão.

Configurações de Controle de Conta de Usuário

O UAC monitora programas em um computador e avisa os usuários quando uma ação pode representar uma ameaça para o sistema. No Windows 7, você pode ajustar o nível de monitoramento que o UAC executa, conforme apresentado na Figura 2. Quando o Windows 7 é instalado, o UAC para a conta principal é configurado no modo: Padrão - Notificar-me somente quando programas tentarem fazer alterações no meu computador

Para ajustar o nível de monitoramento do UAC, use o seguinte caminho:

Iniciar > Painel de Controle > Contas de Usuário > Alterar configurações de Controle de Conta de Usuário
