Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle Exclusivos para Versões Específicas do Windows

Uma série de utilitários do Painel de Controle são exclusivos para o Windows 7, incluindo os seguintes:

  • Grupo Domestico

  • Central de Ações

  • Windows Defender

  • Conexões de RemoteApp e Área de Trabalho

  • Solução de Problemas

Grupo Domestico

Um grupo doméstico é uma configuração de rede que permite aos usuários compartilhar arquivos e pastas facilmente em uma rede doméstica. O utilitário Grupo Doméstico, mostrado na Imagem 1, não está disponível para redes corporativas ou públicas.

Central de Ações

A Central de Ações substitui a Central de Segurança, que se encontra no XP e Vista. A Central de Ações é descrito com mais detalhes na página intitulada Central de Ações, encontrada em outro local deste curso.

Windows Defender

O Windows Defender é um programa anti-spyware que varre o sistema operacional para encontrar softwares indesejados que podem representar uma ameaça à segurança. O Windows Defender protege os computadores contra softwares nocivos, como spyware, que é criado para realizar ataques maliciosos. Se o Windows Defender detectar qualquer software indesejado, os usuários podem excluir ou colocar em quarentena os arquivos nocivos. Para acessar o Windows Defender, use o seguinte caminho:

Iniciar> Painel de Controle> Windows Defender

Windows Defender está disponível no Windows 7 e Windows Vista por padrão e pode ser baixado para Windows XP.

Conexões de RemoteApp e Área de Trabalho

O utilitário Conexões de RemoteApp e Área de Trabalho, mostrado na Imagem 2, fornece acesso a computadores remotos e programas a partir de uma única pasta no Windows Explorer. Para acessar o utilitário Conexões de RemoteApp e Área de Trabalho, use o seguinte caminho:

Iniciar> Painel de Controle> Conexões de RemoteApp e Área de Trabalho

Solução de Problemas

O utilitário de solução de problemas, mostrado na Imagem 3, permite solucionar problemas dentro das seguintes categorias:

  • Programas - Resolva problemas de compatibilidade com programas criados para versões anteriores do Windows.

  • Hardware e som - Diagnostica e soluciona problemas de configuração dos dispositivos e questões de áudio.

  • Rede e Internet - Resolve problemas com conexões de Internet, bem como arquivos e pastas compartilhadas.

  • Aparência e Personalização - Resolve problemas associados com a aparência da área de trabalho.

  • Sistema e Segurança - Executa tarefas de manutenção, verifica se há problemas de desempenho e melhorar o uso de energia.

Para acessar o utilitário de solução de problemas, use o seguinte caminho:

Iniciar> Painel de Controle> Solução de problemas
