Instalação do Sistema Operacional

Procedimentos de Configuração de Disco Rígido

Depois que a instalação do Windows copiar todos os arquivos do sistema operacional necessários para o disco rígido, o computador será reiniciado e solicitará que você crie uma conta de usuário.

O Windows 7 deve ser registrado. Conforme mostrado na Figura 1, você também deve concluir a verificação que garante que você está usando uma cópia legal. Isso permite que você baixe as atualizações individuais, chamadas de patches. Um service pack são vários patches agrupados.

Windows Update

Dependendo da mídia no momento da instalação, pode haver atualizações a serem instaladas. Conforme mostrado na Figura 2, você pode usar o Windows Update, localizado no menu Iniciar, para procurar novos softwares, assim como instalar service packs e patches de atualização.

Para instalar patches e service packs no Windows 7 ou Windows Vista, use o seguinte caminho:

Iniciar> Todos os Programas> Windows Update

Para instalar patches e service packs no Windows XP, use o seguinte caminho:

Iniciar> Todos os Programas> Acessórios> Ferramentas do Sistema> Windows Update

Gerenciador de Dispositivos

Após a instalação, verifique se todo os hardwares estão instalados corretamente. Como mostrado na Figura 3, você pode usar o Gerenciador de Dispositivos para localizar problemas e instalar os drivers corretos e atualizados, no Windows 7 e Windows Vista, usando o seguinte caminho:

Iniciar> Painel de Controle> Gerenciador de Dispositivos

No Windows XP, utilize o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Hardware > Gerenciador de Dispositivos

No Gerenciador de Dispositivos do Windows 7 e Windows Vista, um triângulo amarelo com um ponto de exclamação indica um problema no dispositivo. Para visualizar a descrição do problema, clique com o botão direito do mouse no dispositivo e selecione Propriedades. Um círculo cinza com uma seta apontando para baixo significa que o dispositivo está desativado. Para ativar o dispositivo, clique com o botão direito no dispositivo e selecione Ativar. Para expandir a categoria de dispositivo, clique no triângulo apontando para a direita, localizado ao lado da categoria.

NOTA: No Windows XP, um círculo vermelho com um X branco indica um dispositivo desativado.

NOTA: Quando o Windows detecta um erro do sistema, uma caixa de diálogo é exibida. Se você optar por enviar o relatório, o Microsoft Windows Error Reporting (WER) coletará informações sobre a aplicação envolvida no erro e enviará as informações para a Microsoft.
