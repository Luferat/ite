Técnicas de Manutenção Preventiva para os Sistemas Operacionais

Plano de Manutenção Preventiva para o S.O.

Você pode agendar as aplicações de manutenção preventiva para serem executadas em determinado momento. Você pode agendar essas tarefas utilizando o Agendador de Tarefas (interface GUI) ou através do prompt de comando (interface CLI). Ambas as ferramentas permitem que você execute um comando em um momento específico ou em uma base contínua de dias e horários selecionados. O Agendador de Tarefas do Windows, exibido na figura, apresenta um interface de uso e aprendizagem mais fácil do que o comando at, ideal para agendar tarefas recorrentes ou deletar tarefas agendadas.

Agendador de Tarefas do Windows

Você pode automatizar tarefas utilizando o Agendador de Tarefas. O Agendador de Tarefas monitora os critérios selecionados e definidos pelo usuário e, em seguida, executa as tarefas quando os mesmos forem cumpridos. Algumas tarefas que podem ser automatizadas utilizando o Agendador de Tarefas incluem:

  • Limpeza de Disco

  • Backup

  • Desfragmentador de Disco

  • Ponto de Restauração

  • Iniciar outras aplicações

Para acessar o Agendador de Tarefas no Windows 7 e no Windows Vista, utilize o seguinte caminho:

Iniciar> Todos os Programas> Acessórios> Ferramentas do Sistema> Agendador de Tarefas

No Windows XP, use o seguinte caminho:

Iniciar> Todos os Programas> Acessórios> Ferramentas do Sistema> Tarefas Agendadas

Comando at

Você pode usar o comando at para agendar que um comando, arquivo de script ou aplicativo seja executado em uma data e hora específica. Para utilizar o comando at, você deve estar logado como administrador.

Para obter mais informações sobre o comando at no Windows 7 e Windows Vista, use o seguinte caminho:

Iniciar> no campo Pesquisar programas e arquivos, digite cmd e pressione Enter> digite at/? na linha de comando e pressione Enter

Para obter informações sobre o comando at no Windows XP, use o seguinte caminho:

Iniciar> Executar> digite cmd e pressione Enter> digite at/? na linha de comando e pressione Enter
