Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle Exclusivos para Versões Específicas do Windows

Uma série de utilitários do Painel de Controle são exclusivos para o Windows Vista, incluindo os seguintes:

  • Configurações do Tablet PC

  • Canetas e Dispositivos de Entrada

  • Arquivos Offline

  • Relatórios de Problemas e Soluções

  • Impressoras

Configurações do Tablet PC

O utilitário Configurações do Tablet PC, mostrado na Imagem 1, permite que você personalize a funcionalidade do seu tablet PC. Para acessar o utilitário Configurações do Tablet PC, use o seguinte caminho:

Iniciar> Painel de Controle> Configurações do Tablet PC

Canetas e Dispositivos de Entrada

O utilitário Caneta e Dispositivos de Entrada, mostrado na Imagem 2, permite que você defina as configurações para as canetas digitais e outros dispositivos de entrada. Para acessar o utilitário Caneta e Dispositivos de Entrada, use o seguinte caminho:

Iniciar> Painel de Controle> Caneta e Dispositivos de Entrada

Arquivos Offline

O utilitário Arquivos Offline, mostrado na Imagem 3, permite que você selecione os arquivos e pastas compartilhadas a partir de uma rede a para serem armazenado em um computador. Esses arquivos estarão disponíveis depois que o computador estiver desconectado da rede. Quando você se reconectar à rede, as mudanças que fez offline serão aplicadas aos arquivos originais na rede.

Para configurar um computador para usar arquivos e pastas off-line, use o seguinte caminho:

Iniciar> Painel de Controle> Arquivos Offline> clique na guia Geral> clique em Ativar Arquivos Offline

Para ver uma lista de todos os arquivos off-line, use o seguinte caminho:

Iniciar> Painel de Controle> Arquivos Offline> clique na guia Geral> clique em Exibir arquivos offline

Relatórios de Problemas e Soluções

O utilitário Relatórios de Problemas e Soluções, mostrado na Imagem 4, mantém um registro de problemas ocorridos no sistema e soluções que foram fornecidas pela Microsoft. Este utilitário foi transferido para a Central de Ações no Windows 7.

Para acessar o utilitário Relatórios de Problemas e Soluções, use o seguinte caminho:

Iniciar> Painel de Controle> Relatórios de Problemas e Soluções

Impressoras

O utilitário Impressoras, como mostrado na Imagem 5, permite que você adicione, remova e configure impressoras. Para acessar o utilitário Impressoras, use o seguinte caminho:

Iniciar> Painel de Controle> Impressoras
