Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

O Gerenciador de Dispositivos, como mostrado na Figura 1, exibe o hardware presente em um computador. O Gerenciador de Dispositivo permite que você diagnostique e resolva conflitos de dispositivos. Você pode visualizar detalhes sobre o hardware e os drivers instalados, assim como executar as seguintes funções:

  • Atualizar driver - Atualize o driver instalado atualmente.

  • Reverter driver - Reverta o driver instalado atualmente por outro instalado anteriormente.

  • Desinstalar driver - Remova o driver.

  • Desativar dispositivo - Desativa o dispositivo.

Para acessar o Gerenciador de Dispositivos no Windows 7 e Windows Vista, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Gerenciador de Dispositivos

No Windows XP, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Hardware > Gerenciador de Dispositivos

Você pode visualizar as propriedades de qualquer dispositivo no sistema, clicando duas vezes no nome do dispositivo.
