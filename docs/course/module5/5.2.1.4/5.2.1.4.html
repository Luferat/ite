Instalação do Sistema Operacional

Procedimentos de Configuração de Disco Rígido

Quando um computador é inicializado com o disco de instalação do Windows 7 (ou uma unidade USB), o assistente de instalação apresenta três opções, como mostrado na Figura 1:

  • Instalar agora - Configura e instala o Windows 7.

  • O que saber antes de instalar o Windows - Abre as janelas Ajuda e Suporte que descrevem a atualização e opções personalizadas de instalação do Windows 7. A janela também descreve como preparar e instalar o Windows 7.

  • Reparar o computador - Abre o utilitário Opções de Recuperação do Sistema para reparar uma instalação. Selecione a instalação do Windows 7 que precisa de reparos e clique em Avançar. Você pode então selecionar a partir de uma série de ferramentas de recuperação, como o Reparo de Inicialização. O Reparo de Inicialização localiza e repara problemas com os arquivos do sistema operacional. Se o reparo de inicialização não resolver o problema, opções adicionais, como a Restauração do Sistema e a Recuperação de Imagem do Sistema, estão disponíveis.

NOTA: Antes de executar uma instalação de reparo, faça um backup dos arquivos importantes em um local diferente, como em um segundo disco rígido, em um disco óptico ou em um dispositivo de armazenamento USB.

Para esta seção, selecione a opção Instalar agora. Três opções estão disponíveis:

  • Atualizar - Atualiza o Windows, porém mantém seus arquivos atuais, configurações e programas. Você pode usar esta opção para reparar uma instalação.

  • Personalizado (avançado) - Instala uma cópia limpa do Windows em sua escolha de localização e permite que você mude os discos e partições. Também conhecida como instalação limpa. Selecionar a instalação personalizada aumenta a probabilidade de uma instalação bem-sucedida.

  • Sair - Sai do Setup.

Se as instalações existentes do Windows não forem encontradas, a opção de Atualização será desativada, conforme mostrado na Figura 2.

NOTA: A menos que você execute uma instalação limpa do Windows 7, as pastas anteriores do Windows serão mantidas, juntamente com a pasta Documents and Settings e pastas Arquivos de Programas. Durante a instalação do Windows 7, essas pastas são movidas para uma pasta chamada Windows.old. Você pode copiar os arquivos da instalação anterior para a nova instalação se você precisar deles.

Durante a instalação, você deve fornecer as seguintes informações:

  • Idioma a ser instalado

  • Padrões e formatos que definem moedas e números

  • Teclado ou método de entrada

  • Localização física da instalação

  • Usuário e nome do computador

  • Senha para a conta de administrador

  • Chave do produto

  • Configurações de data e hora

  • Configurações de rede

Configurações de Rede

Ao configurar as definições iniciais de rede durante a instalação, você será solicitado a escolher um dos seguintes locais atuais, como mostrado na Figura 3:

  • Rede Doméstica

  • Rede de Corporativa

  • Rede Pública

Dependendo da localização atual do computador e da versão do sistema operacional, você será solicitado para selecionar um método para organizar os computadores e compartilhar recursos em uma rede. As opções são Grupo Doméstico, Grupo de Trabalho e Domínio.

Se a opção Rede Doméstica for selecionada, será solicitado que você digite o nome de um grupo de trabalho, juntamente com a opção de configurar um grupo doméstico. Um grupo de trabalho fornece uma estrutura de rede que permite o compartilhamento de arquivos e impressora. Todos os computadores em um grupo de trabalho devem ter o mesmo nome do grupo. Um grupo doméstico permite que os computadores na mesma rede compartilhem automaticamente arquivos, como músicas e fotos, bem como impressoras.

Se a opção Rede Corporativa for selecionada, você pode optar por digitar o nome de um domínio ou grupo de trabalho. Um computador em um domínio é gerenciado por um administrador central e deve seguir as regras e procedimentos definidos pelo administrador. Um domínio, como um grupo de trabalho, oferece aos usuários a capacidade de compartilhar arquivos e dispositivos.
