Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle

Garantir o acesso adequado aos arquivos requer a gestão do diretório e as configurações de pasta. Para definir as configurações de pastas no Windows, use o utilitário Opções de Pasta.

Para acessar o utilitário Opções de Pasta no Windows 7, use o seguinte caminho:

Iniciar > Painel de Controle > Opções de Pasta

Opções de Pasta, conforme apresentado na imagem, possuem três guias.

Guia Geral

Utilize a guia Geral para ajustar a exibição básica e as configurações de acesso.

  • Procurar pastas - Configura como uma pasta é exibida quando for aberta.

  • Clicar nos itens da seguinte maneira - Especifica o número de cliques necessários para abrir um arquivo.

  • Painel de navegação - Determina se todas as pastas são apresentadas e se a pasta é automaticamente expandida quando ela é selecionada no painel de navegação.

Guia Modo de Exibição

Use a guia Exibir para ajustar as configurações de exibição e atributos para as pastas, incluindo a capacidade de ver pastas ocultas.

  • Modo de exibição - Aplica as configurações de exibição para uma pasta que está sendo visualizada a todas as pastas do mesmo tipo.

  • Configurações avançadas - Personaliza as experiências de visualização.

Guia Pesquisar

Use a guia Pesquisar para personalizar as configurações de busca das pastas.

  • O que pesquisar - Configurações de busca baseados em locais indexados e não indexados para fazer os arquivos e pastas mais fáceis de encontrar.

  • Como pesquisar - Determina quais opções levar em conta durante a pesquisa.

  • Ao pesquisar locais não indexados - Determina se os diretórios do sistema e os arquivos compactados são incluídos na pesquisa locais não indexados.
