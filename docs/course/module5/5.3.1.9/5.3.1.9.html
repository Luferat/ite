Desktop, Ferramentas e Aplicações

A biblioteca é um novo recurso do Windows 7, que permite organizar facilmente o conteúdo de vários dispositivos de armazenamento no computador local e de rede, incluindo a mídia removível, sem realmente mover os arquivos. Uma biblioteca é uma pasta virtual que apresenta conteúdos de diferentes locais dentro do mesmo ponto de exibição. Você pode realizar pesquisas dentro da biblioteca, filtrando o conteúdo utilizando critérios como nome do arquivo, tipo de arquivo ou data de modificação.

Quando o Windows 7 está instalado, cada usuário possuí quatro bibliotecas padrões: Documentos, Músicas, Imagens e Vídeos. Para acessar a biblioteca, abra o Windows Explorer e clique em Bibliotecas, localizada na coluna da esquerda. Para adicionar um arquivo ou pasta na biblioteca, clique com o botão direito do mouse, selecione Incluir na biblioteca e, em seguida, escolha em qual biblioteca você irá adicionar o item. O arquivo ou pasta estará disponível quando você abrir a biblioteca.

Para criar uma nova biblioteca, abra uma pasta e selecione Bibliotecas > Nova biblioteca.

Para personalizar uma biblioteca, clique com o botão direito do mouse na biblioteca e, em seguida, clique em Propriedades. A janela Propriedades permite que você adicione pastas à biblioteca clicando em Incluir uma pasta. Você também pode alterar o ícone da biblioteca e personalizar a forma como os itens são organizados.
