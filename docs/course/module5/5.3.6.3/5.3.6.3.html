Interface Gráfica do Windows e Painel de Controle

Utilitários do Painel de Controle Exclusivos para Versões Específicas do Windows

Uma série de utilitários do Painel de Controle são exclusivos para Windows XP, incluindo os seguintes:

  • Adicionar/Remover Programas

  • Impressoras e Aparelhos de Fax

  • Atualizações Automáticas

  • Conexões de Rede

  • Assistente de Configuração de Rede

Adicionar/Remover Programas

O utilitário Adicionar/Remover Programas, exibido na Figura 1, oferece a mesma funcionalidade no Windows XP que o utilitários Programas e Recursos oferece para o Windows 7 e Windows Vista. Para acessar a ferramenta Adicionar/Remover Programas no Windows XP, use o seguinte caminho:

Iniciar> Painel de Controle> Adicionar/Remover Programas

Impressoras e Aparelhos de Fax

O utilitário Impressoras e Aparelhos de Fax do Windows XP, mostrada na Figura 2, permite que você adicione impressoras e configure um fax. Funcionalidade semelhante é fornecida pelo utilitário Dispositivos e Impressoras, presente no Windows 7 e Windows Vista. Para acessar o utilitário Impressoras e Aparelhos de Fax, use o seguinte caminho:

Iniciar> Painel de Controle> Impressoras e Aparelhos de Fax

Atualizações Automáticas

O utilitário Atualizações Automáticas, mostrado na Figura 3, garante que o sistema operacional e os programas sejam constantemente atualizados, para fins de segurança e funcionalidades adicionais. O utilitário Atualizações Automáticas verifica o sistema em buscas de atualizações necessárias e, em seguida, recomenda o que deve ser baixado e instalado. É possível fazer o download e instalar as atualizações assim que elas estiverem disponíveis, ou baixar as atualizações, conforme necessário, e instalá-las na próxima vez que o computador for reiniciado. Para acessar o utilitário Atualizações Automáticas, use o seguinte caminho:

Iniciar> Painel de Controle> Sistema> Atualizações Automáticas

Conexões de Rede

O utilitário Conexões de Rede, mostrado na Figura 4, permite ativar e desativar conexões de rede. Foi substituído no Windows 7 e no Windows Vista pela Central de Rede e Compartilhamento. Para acessar o utilitário de Conexões de Rede, use o seguinte caminho:

Iniciar> Painel de Controle> Conexões de Rede

Assistente de Configuração de Rede

O Assistente de Configuração de Rede, mostrado na Figura 5, orienta você durante a instalação da rede de um pequeno escritório ou de uma rede doméstica. Esse assistente permite que os usuários da rede compartilhem arquivos, pastas e dispositivos, como impressoras.

Para acessar o Assistente de Configuração de Rede, use o seguinte caminho:

Iniciar> Painel de Controle> Conexões de Rede> Tarefas comuns e, em seguida, clique em Assistente de Configuração de Rede
