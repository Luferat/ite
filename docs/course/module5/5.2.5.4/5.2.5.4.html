Instalação do Sistema Operacional

Estrutura de Diretório e Atributos de Arquivos

Para visualizar ou alterar as propriedades de uma aplicação, arquivo ou pasta, clique com o botão direito no ícone e selecione Propriedades.

Propriedades do Aplicação e do Arquivo

A janela de Propriedades de uma aplicação ou arquivo, conforme apresentado na Figura 1, contém as seguintes guias:

  • Geral - Exibe informações básicas, incluindo o local e os atributos.

  • Segurança - Oferece a opção de alterar as permissões de acesso a arquivos para contas de usuário e do sistema.

  • Detalhes - Exibe informações básicas sobre o arquivo, incluindo os atributos.

  • Compatibilidade - Fornece opções para configurar o modo de compatibilidade de arquivos e configurações operacionais. No Windows 7, o modo de compatibilidade permite que um usuário execute um programa criado para versões anteriores do sistema operacional Windows. Para o Windows Vista e Windows XP, o número de opções disponíveis no modo de compatibilidade é limitada.

Propriedades da Pasta

A visualização Propriedades de uma pasta, conforme apresentado na Figura 2, contém as seguintes guias:

  • Geral - Exibe informações básicas, tais como local e tamanho. Oferece opções para alterar os atributos, como criar uma pasta somente leitura ou oculta.

  • Compartilhamento - Exibe opções para o compartilhamento da pasta. Os usuários podem compartilhar pastas com computadores da mesma rede. Configurações de proteção por senha também podem ser realizadas.

  • Segurança - Exibe opções para configuração de segurança básica e avançada.

  • Versões Anteriores - Exibe opções para restaurar a pasta a partir de uma versão anterior.

  • Personalizado - Exibe opções para personalizar a aparência da pasta e otimizá-la para determinados tipos de arquivos, como arquivos de música ou foto.

Shadow Copy (Cópia de Sombra)

A Cópia de Sombra, Shadow Copy, é uma característica dos sistemas operacionais Windows que cria automaticamente cópias de backup de arquivos e dados em um disco rígido. Esse recurso pode ser encontrado na guia Versões Anteriores e muitas vezes é referido como o recurso Versões Anteriores. A Cópia de Sombra requer que o disco rígido seja formatado em NTFS e funcione em conjunto com o Backup do Windows e a Restauração do Sistema. Antes que a Cópia de Sombra possa ser utilizada, ela deve ser habilitada. Para habilitar a Cópia de Sombra, use o seguinte caminho:

Iniciar > Painel de Controle > Sistema > Link Proteção do Sistema > Clique na caixa de seleção próxima às unidades desejadas > Clique em OK.

Depois que a Cópia de Sombra foi ativada, os usuários podem visualizar, copiar ou restaurar versões anteriores do arquivo. Selecionar a opção de visualização irá abrir o arquivo como somente leitura, enquanto a opção de cópia irá salvar uma versão mais antiga do arquivo em uma outra pasta. A opção Restaurar irá substituir o arquivo atual.
