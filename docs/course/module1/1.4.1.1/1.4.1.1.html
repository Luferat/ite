Introdução ao Computador Pessoal

Resumo

Este capítulo apresenta os componentes que compõem um sistema de computador pessoal e como considerar a atualização dos componentes. Grande parte do conteúdo deste capítulo irá ajudá-lo ao longo deste curso.

  • Tecnologia da informação engloba o uso de computadores, hardware de rede e software para processar, armazenar, transmitir e recuperar informações.

  • Um sistema de computador pessoal é composto de componentes de hardware e aplicativos de software.

  • O gabinete do computador e a fonte de alimentação devem ser escolhidas cuidadosamente para suportar o hardware no interior do gabinete e para permitir a adição de componentes.

  • Os componentes internos do computador são selecionados para características e funções específicas. Todos os componentes internos devem ser compatíveis com a placa-mãe.

  • Use o tipo correto de portas e cabos quando conectar os dispositivos.

  • Os dispositivos de entrada mais comuns incluem o teclado, mouse, tela sensível ao toque e câmeras digitais.

  • Os dispositivos de saída mais comuns incluem monitores, impressoras e alto-falantes.

  • Gabinetes, fontes de alimentação, CPU e sistema de refrigeração, RAM, discos rígidos e placas de expansão, devem ser atualizados quando os dispositivos falharem ou não atenderem às necessidades dos clientes.

  • Computadores especializados requerem hardware específico para sua função. O tipo de hardware utilizado em computadores especializados é determinado pela forma como o cliente trabalha e o que um cliente deseja realizar.
