Computer to Network Connection

Wireless and Wired Router Configurations

When the router has established communication with the modem, configure the router to communicate with the devices on the network. Open a web browser. In the Address field, enter 192.168.1.1. This is the default private IP address for the Linksys E2500 router configuration and management interface.

The first time you connect to the Linksys E2500, you are asked to either install the Cisco Connect software or connect to the router in an open and unsecured network. When connecting to an open and unsecured E2500 router, a security window, as shown in Figure 1, prompts for authentication to access the router configuration screens. The username field must be left blank. Enter admin as the default password.
