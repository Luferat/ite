Montagem de Computadores

Instalação dos Cabos

Após encaixar os painéis do gabinete, conecte os cabos na parte traseira do computador. Aqui estão alguns exemplos de conexões comuns:

  • Monitor

  • Teclado

  • Mouse

  • USB

  • Ethernet

  • Energia

NOTA: Conecte o cabo de alimentação depois de conectar todos os outros cabos.

Ao conectar os cabos, verifique se eles estão encaixados nos locais corretos. Por exemplo, sistemas mais antigos usam o mesmo tipo de conector PS/2 para conectar os cabos do mouse e do teclado, porém eles são identificados por cores para evitar que o dispositivo seja conectado de forma incorreta. Geralmente, um ícone utilizado para identificar o dispositivo é exibido próximo ao conector.

CUIDADO: Ao ligar os cabos, nunca force uma conexão.

Para instalar os vários cabos externos, siga esses passos:

Passo 1. Conecte o cabo do monitor à porta de vídeo (Figura 1). Prenda o cabo apertando os parafusos do conector.

Passo 2. Conecte o cabo do teclado na porta teclado PS/2 (Figura 2).

Passo 3. Conecte o cabo do mouse na porta mouse PS/2 (Figura 3).

Passo 4. Conecte o cabo USB em uma porta USB (Figura 4).

Passo 5. Conecte o cabo de rede na porta de rede (Figura 5).

Passo 6. Conecte a antena wireless no conector da placa de rede (Figura 6).

Passo 7. Conecte o cabo de alimentação na fonte de energia (Figura 7).

NOTA: Algumas placas-mãe não possuem portas PS/2 para conectar o teclado e mouse. Conecte um teclado USB e um mouse USB neste tipo de placa-mãe.
