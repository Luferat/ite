Instalação dos Drives

As unidades que estão instaladas nos compartimentos internos são chamadas de unidades internas. O disco rígido (HDD) é um exemplo de unidade interna.

Para instalar o disco rígido, siga esses passos:

Passo 1. Posicione o disco rígido para que ele se alinhe com a baia que possui uma abertura de 3,5 polegadas (8,9 centímetros).

Passo 2. Insira o disco rígido no compartimento da unidade de forma que os orifícios dos parafusos da unidade se alinhem com os orifícios do gabinete.

Passo 3. Fixe a unidade no gabinete usando os parafusos adequados.
