Inicializando o Computador

Configuração da BIOS

A BIOS pode suportar muitos recursos de segurança para proteger suas configurações e os dados no disco rígido, além de também ajudar a recuperar o computador em caso de roubo. Existem vários recursos comuns de segurança encontrados na BIOS.

Senhas da BIOS - As senhas permitem diferentes níveis de acesso às configurações da BIOS, conforme apresentado na figura.

  • Senha Supervisor - Esta senha pode acessar todas as senhas de acesso de usuários e todas as telas e configurações da BIOS.

  • Senha Usuário - Esta senha torna-se disponível após a senha de supervisor estiver habilitada. Use essa senha para definir o nível de acesso aos usuários.

Estes são alguns níveis comuns de acesso de usuário:

  • Acesso Total - Todas as telas e configurações estão disponíveis, exceto a configuração da senha de supervisor.

  • Limitado - Apenas algumas mudanças e configurações podem ser feitas, por exemplo, a data e hora.

  • Visualização Apenas - Todas as telas estão disponíveis, mas nenhuma configuração pode ser alterada.

  • Sem acesso - Nenhum acesso é fornecido ao utilitário de configuração da BIOS.

Criptografia de Unidade - O disco rígido pode ser criptografado para evitar o roubo de dados. A criptografia altera os dados em código que não podem ser entendidos. Sem a senha correta, o computador não inicia e também não descriptografa os dados. Mesmo que o disco rígido seja colocado em outro computador, os dados criptografados permanecem criptografados.

Trusted Platform Module - O chip TPM contém itens de segurança, tais como chaves de criptografia e senhas.

Lojack - Este é um sistema de duas partes utilizados para proteger os computadores da Absolute Software. A primeira parte é um programa chamado de Módulo de Persistência que está instalado na BIOS pelo fabricante. A segunda parte é um programa chamado Agente de Aplicação que é instalado pelo usuário. Quando o Agente de Aplicação é instalado, o Módulo de Persistência é ativado. O Módulo Persistência instala o Agente de Aplicação caso ele seja removido do computador. O Módulo de Persistência não pode ser desligado após ser ativado. O Agente de Aplicação chama o Centro de Monitoramento Absoluto pela Internet para relatar informações sobre o dispositivo e localização em um cronograma definido. Se o computador for roubado, o proprietário pode entrar em contato Absolute Software e realizar as seguintes funções:

  • Bloquear o computador remotamente.

  • Exibe uma mensagem de modo que o computador perdido possa ser devolvido para o proprietário.

  • Excluir os dados confidenciais do computador.

  • Localiza o computador utilizando geotecnologia.
