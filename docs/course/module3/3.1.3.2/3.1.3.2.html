Instale os Drives

As unidades ópticas armazenam dados em mídias removíveis, como CDs, DVDs e Blu-rays. As unidades ópticas são instaladas em baias que são acessados ​​pela parte frontal do gabinete. As baias externas permitem o acesso à algumas mídias, sem abrir o gabinete.

Um conector de alimentação Molex ou SATA pode fornecer energia de uma fonte de alimentação a unidade óptica. Um cabo de dados PATA ou SATA conecta a unidade óptica à placa-mãe.

Para instalar uma unidade óptica, siga estes passos:

Passo 1. Posicione a unidade óptica para que ela se alinhe com a baia com abertura de 5,25 polegadas (13,34 centímetros).

Passo 2. Insira a unidade óptica no compartimento da baia para que os orifícios dos parafusos da unidade óptica se alinhem com os orifícios do gabinete.

Passo 3. Fixe a unidade óptica no gabinete, utilizando os parafusos adequados.
