Atualizando e Configurando um PC

Placa-mãe e seus Componentes

Uma forma de aumentar o desempenho de um computador é aumentar a velocidade de processamento. Você pode fazer isso trocando a CPU. No entanto, você deve atender alguns requisitos.

  • A nova CPU deve caber no soquete da CPU existente.

  • A nova CPU deve ser compatível com o chipset da placa-mãe.

  • A nova CPU deve funcionar com a placa-mãe e a fonte de alimentação existente.

  • A nova CPU deve operar com a memória RAM existente. A memória RAM talvez precise ser atualizada ou expandida para obter o máximo de proveito da nova CPU.

Se a placa-mãe for antiga, você pode não encontrar uma CPU compatível. Nesse caso, você deve substituir a placa-mãe.

CUIDADO: Sempre trabalhe com uma manta e uma pulseira antiestática durante a instalação e remoção de CPUs. Coloque a CPU na manta antiestática até que você esteja pronto para usá-la. Armazene a CPU um uma embalagem antiestática.

Para trocar a CPU, retire a CPU existente, liberando-a do soquete, usando a alavanca de força de inserção zero. Diferentes soquetes têm mecanismos um pouco diferentes, mas todos servem para bloquear a CPU no lugar e orientá-la corretamente no soquete.

Insira o nova CPU. Não force a CPU em seu soquete ou use força excessiva para fechar as barras de travamento. A força excessiva pode danificar a CPU ou o seu soquete. Se você encontrar resistência, certifique-se de que você alinhou a CPU corretamente. A maioria tem um padrão de pinos que se encaixam apenas de uma forma:

  • Soquete SEC - Alinhe os encaixes na CPU para as chaves no soquete SEC.

  • Soquete PGA, LIF e ZIF - Alinhe a CPU ao indicador de conexão 1, que está alinhado com o pino 1 no soquete da CPU.

  • Socket LGA - Alinhe a CPU para que as duas ranhuras se encaixam nas duas extensões do soquete.

A nova CPU pode exigir um dissipador de calor e uma ventoinha diferente. O dissipador e a ventoinham devem acomodar fisicamente a CPU e devem ser compatíveis com o soquete do processador. Eles também devem remover, de forma adequada, o calor gerado pela nova CPU.

ATENÇÃO: Você deve aplicar pasta térmica entre o novo processador e o dissipador de calor.

Veja as configurações térmicas na BIOS para determinar se há algum problema com a CPU, o dissipador de calor e a ventoinha. Aplicativos de terceiros também podem relatar informações sobre a temperatura da CPU em um formato fácil de ler. Consulte a placa-mãe ou a documentação de usuário da CPU para determinar se o chip está operando na faixa de temperatura correta.

Para instalar ventiladores adicionais no gabinete para ajudar a resfriar a placa-mãe e a CPU, siga estes passos:

Passo 1. Alinhe o ventilador na direção correta, quer seja inserir o ar para dentro do gabinete ou soprar o ar para fora.

Passo 2. Prenda o ventilador utilizando os furos do gabinete. Para puxar o ar para dentro do gabinete, monte o ventilador na parte inferior do gabinete. Para direcionar o ar quente para fora, monte o ventilador no topo do gabinete.

Passo 3. Conecte os ventiladores à fonte de alimentação ou a placa-mãe, dependendo do modelo.
