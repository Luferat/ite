Inicializando o Computador

POST e BIOS

Quando o computador é iniciado, o sistema básico de entrada/saída (BIOS) executa uma verificação básica do hardware. Esta verificação é chamada de power-on self-test (POST).

O POST, conforme apresentado na Imagem 1, verifica se o hardware do computador está funcionando corretamente. Se um dispositivo não está funcionando corretamente, um erro ou um código de bipe alerta o técnico do problema. Normalmente, um único bipe significa que o computador está funcionando corretamente. Se houver um problema de hardware, uma tela em branco pode aparecer na inicialização, e o computador emite uma série de sinais sonoros. Cada fabricante de BIOS usa códigos diferentes para indicar problemas de hardware. A Imagem 2 mostra um gráfico dos códigos de bipes. Os códigos de bipes podem ser diferentes para o seu computador. Consulte a documentação da sua placa-mãe para obter os códigos de bipes para o seu computador.

POST Card

Ao verificar um problema no computador quando o vídeo não está disponível, você pode usar um POST card. O POST card é instalado em uma porta na placa-mãe, tais como PCI ou PCIe. Quando o computador é iniciado e alguns erros são encontrados, o PC emite um código que é exibido no POST card. Este código é utilizado para diagnosticar a causa do problema através da placa-mãe, BIOS ou do POST card do fabricante.
