Inicializando o Computador

POST e BIOS

A BIOS contém um programa utilizado para configurar as definições dos dispositivos de hardware. Os dados de configuração são salvos em um chip de memória chamado de Complementary Metal Oxide Semiconductor (CMOS), conforme apresentado na Figura 1. A CMOS é mantida pela bateria do computador. Se a bateria descarregar, os dados de configuração da BIOS serão perdidos. Caso isso aconteça, substitua a bateria e reconfigure a BIOS.

Muitas placas-mãe modernas usam a memória não volátil para armazenar as configurações da BIOS. Este tipo de memória não precisa de energia para armazenar as configurações. Nesse caso, a bateria é utilizada apenas para armazenar a hora e data correta no relógio. As configurações da BIOS não são perdidas quando a bateria descarrega ou é removida.

Para acessar o programa de configuração da BIOS, pressione a tecla ou a sequência de teclas apropriadas durante o POST. Muitas placas-mãe exibem imagens enquanto o computador passa pelo processo de POST. Assim, o computador pode não exibir a tecla ou a sequência de teclas necessárias. A maioria dos computadores usam a tecla DEL ou uma tecla de função para acessar a BIOS. Consulte o manual da placa-mãe para descobrir a tecla ou sequência de teclas utilizada pelo seu computador.

A Figura 2 apresenta o exemplo de um programa de configuração da BIOS. Estas são algumas opções presentes no menu de configuração da BIOS:

  • Main - Configurações básica do sistema

  • Advanced - Configurações avançadas do sistema

  • Boot - Opções de dispositivos e ordem de inicialização

  • Segurança - Configurações de segurança

  • Power - Configurações de gerenciamento avançado de energia

  • JUSTw00t! - Configurações avançadas de tensão e clock

  • Exit - Opções de saída da BIOS e para carregar as configurações padrão
