Montagem de Computadores

Instalação da Placa-Mãe

A placa-mãe está pronta para ser instalada no gabinete do computador, conforme apresentado na Imagem 1. Espaçadores (standoffs) de plástico e de metal são usados ​​para montar a placa e para impedi-la de tocar as porções metálicas do gabinete. Instale apenas os pinos que se alinham com os furos na placa-mãe. A instalação de suportes adicionais pode impedir que a placa-mãe seja encaixada corretamente no gabinete do computador.

Como as conexões de I/O (entrada e saída) de cada placa-mãe podem variar, uma placa de I/O, conforme apresentado na Imagem 2, é fornecida com a placa-mãe. A placa de I/O possui recortes específicos na parte traseira para os conectores. A placa de I/O é instalada na parte traseira do gabinete do computador para permitir que os conectores possam ser utilizados quando a placa-mãe é instalada no gabinete.

Para instalar a placa-mãe, siga estes passos:

Passo 1. Instale os espaçadores no gabinete do computador alinhado os mesmos com as posições dos furos de montagem na placa-mãe.

Passo 2. Instale a placa de I/O na parte traseira do gabinete.

Passo 3. Alinhe os conectores de I/O da parte traseira da placa-mãe com as aberturas na placa de I/O.

Passo 4. Alinhe os orifícios dos parafusos da placa-mãe com os espaçadores.

Passo 5. Insira todos os parafusos da placa-mãe.

Passo 6. Aperte todos os parafusos da placa-mãe.
