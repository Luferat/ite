Instalação da Placa-Mãe

A RAM pode ser instalada na placa-mãe antes da mesma ser presa no gabinete do computador. Antes da instalação, consulte a documentação da placa-mãe ou o site do fabricante para garantir que a memória RAM é compatível com a mesma.

A RAM fornece um armazenamento rápido dos dados temporários para a CPU enquanto o computador estiver em funcionamento. A RAM é a memória volátil, o que significa que seu conteúdo é perdido quando o computador é desligado. Normalmente, o aumento de memória RAM melhora o desempenho do computador.

Para instalar a RAM, siga estes passos:

Passo 1. Alinhe os encaixes do módulo de memória RAM com as chaves do slot e pressione para baixo até que as abas laterais sejam encaixadas.

Passo 2. Certifique-se de que as abas laterais tenham travado o módulo da memória RAM.

Passo 3. Verifique visualmente por contatos expostos.

Repita estas etapas para os módulos adicionais.
