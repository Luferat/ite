Atualizando e Configurando um PC

Placa-mãe e seus Componentes

Os sistemas de computadores precisam de atualizações periódicas, por várias razões:

  • Os requisitos do usuário mudaram.

  • Novos softwares exigem um novo hardware.

  • O novo hardware oferece desempenho aprimorado.

Alterações no computador podem fazer com que você atualize ou substitua componentes e periféricos. Pesquise e avalie o custo benefício, tanto para atualização, quanto para substituição.

Se você atualizar ou substituir a placa-mãe, pode ser necessário que você substitua outros componentes, como o processador, dissipador de calor, ventoinha e a memória RAM. A nova placa-mãe deve se encaixar no gabinete do computador antigo. A fonte de alimentação deve ser compatível com a nova placa-mãe e também suportar todos os novos componentes do computador.

Comece a atualização movendo a CPU, dissipador de calor e ventoinha para a nova placa-mãe. É mais fácil trabalhar com estes equipamentos quando eles estão fora do gabinete. Trabalhe em uma manta antiestática e use a pulseira antiestática para evitar danificar a CPU. Se a nova placa-mãe requer um modelo diferente de CPU e memória RAM, instale-os nesse momento. Limpe a pasta térmica do processador e do dissipador de calor. Lembre-se de adicionar a pasta térmica entre o processador e o dissipador de calor.

Instalação da CPU

Diferentes arquiteturas de CPU são instaladas nestes soquetes:

  • Single-Edge Connector (SEC)

  • Low-Insertion Force (LIF)

  • Zero-Insertion Force (ZIF)

  • Land Grid Array (LGA)

  • Pin Grid Array (PGA)

Soquetes SEC e LIF não são mais utilizados. Consulte a documentação da placa-mãe sobre como instalar a CPU.

Configurações de Jumper

Os jumpers são os pinos dourados que se encontram em pé na placa-mãe. Cada agrupamento de dois ou mais pinos é chamado de bloco de jumper. A placa-mãe pode usar um switch Dual In-line Package (DIP), ao invés dos jumpers. Ambos os métodos são usados ​​para completar circuitos elétricos, que fornecem uma variedade de opções suportadas pela placa-mãe. A documentação da placa-mãe indica quais pinos devem ser conectados para acomodar as várias opções:

  • Voltagem da CPU

  • Velocidade da CPU

  • Velocidade do barramento

  • Tamanho e Tipo de Cache

  • Flash BIOS habilitado

  • Limpar CMOS

  • Tamanho da Memória do Sistema

As placas-mãe mais atuais raramente possuem jumpers. Tecnologias eletrônicas avançadas permitem que estas opções sejam configuradas a partir do programa de configuração da BIOS.

Instalação da Bateria da CMOS

A bateria da CMOS pode precisar ser substituída depois de alguns anos. Se o computador não estiver armazenando a hora e data correta do sistema durante as vezes que o mesmo é desligado, provavelmente a bateria se esgotou. Certifique-se de que a nova bateria corresponde ao modelo exigido pela placa-mãe.

Para instalar a bateria da CMOS, siga estes passos:

Passo 1. Com cuidado, deslize para o lado, ou puxe para cima, os clipes de metal para remover a bateria antiga.

Passo 2. Alinhe os pólos positivos e negativos na posição correta.

Passo 3. Com cuidado, deslize para o lado, ou puxe para cima, os clipes de metal para inserir a nova bateria.
